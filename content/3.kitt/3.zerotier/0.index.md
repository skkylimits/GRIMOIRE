## Install ZeroTier:
   - [Download](https://www.zerotier.com/download/) and install on all machines.

## Join a Network:
   - **Option 1 (Terminal):**  
     Run on each machine:
     ```shell [terminal]
     zerotier-cli join <network-id>
     ```
   - **Option 2 (Manual):**  
     Right-click the ZeroTier tray icon → Select **Join Network** → Enter your Network ID.

## Authorize Devices:
   - Approve devices in the ZeroTier web portal.

## Verify Connection:
   - Ping between devices using ZeroTier-assigned IPs to confirm connectivity.
