## Windows -> Linux

Install xrdp

- sudo su
- apt install xrdp
- update -rc.d xrdp enable (enable on startup)
- service xrdp start
- service xrdp status

Create user account

> In linux you can't remote into a logged in user session that's you're already locked in with.
- adduser --home /skkylimits skkylimits
- usermod -aG sudo skkylimits

Log into user and sudo su to verify account is created then switch user again.

Windows RDP > linux

Open remote desk, fill in the ip of <target machine>
	- enter username
	- enter password
