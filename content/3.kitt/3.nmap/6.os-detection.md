## 🖥️ Operating System Detectie

### OS fingerprinting

```bash
nmap -O target
```

**Requirements:**
- Minimaal 1 open poort
- Minimaal 1 closed poort

### OS Scan Limit

```bash
nmap -O --osscan-limit target
```
::note
Slaat OS-detectie automatisch over wanneer:

* Er **te weinig poorten** open/closed zijn
* De host **te weinig fingerprint-data** teruggeeft

Voorkomt valse of onbetrouwbare OS-guesses

Gebruik `--osscan-limit` wanneer je **snellere scans** wilt en valse OS-resultaten wilt vermijden, vooral op gefilterde of streng beveiligde systemen.
::

### OS guess (aggressive)

```bash
nmap -O --osscan-guess target
nmap -O --fuzzy target           # Synonym
```

**Example output:**
```
Running: Linux 5.X
OS CPE: cpe:/o:linux:linux_kernel:5
OS details: Linux 5.0 - 5.4
```

::note
OS detection werkt door TCP/IP stack fingerprinting - verschillen in implementaties tussen OS'en.
::
