## 🔌 Ports & Port Scanning

### Fast mode

```bash
nmap -F target
```

→ Top 100 meest voorkomende poorten.

### Specifieke poorten scannen

**Eén poort:**
```bash
nmap -p 80 target
```

**Meerdere:**
```bash
nmap -p 22,80,443 target
```

**Range:**
```bash
nmap -p 1-500 target
```

**Service-namen:**
```bash
nmap -p http,ssh target
```

**Alle poorten:**
```bash
nmap -p- target
```

::important
`-p-` scant alle 65535 poorten - dit is **zeer traag**!
::

### UDP + TCP samen

```bash
nmap -sU -sT -p U:53,T:25 target
```

### Top Ports

```bash
nmap --top-ports 10 target      # Top 10
nmap --top-ports 100 target     # Top 100
```

### Sequential scanning

```bash
nmap -r target
```

::note
Standaard scant Nmap ports in random order (firewall evasion). `-r` schakelt dit uit.
::


### **Real-time scanning**

```bash
nmap -v -r target
```

`-v` toont real-time scanprogressie tijdens het scannen.
