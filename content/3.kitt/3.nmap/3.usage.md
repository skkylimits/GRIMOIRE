## 🎯 De Kunst van Digitale Recon

### Belangrijke help-opties

```bash
nmap -h              # Help summary
man nmap             # Complete manual
nmap -V              # Versie bekijken
```

**Verbose output:**
```bash
nmap -v target       # Verbose
nmap -vv target      # Extra verbose
```

### Een simpele scan

```bash
nmap target
```

Scant de **top 1000 TCP poorten**.

### Port States

Nmap kan een poort in 6 statussen plaatsen:

| State          | Betekenis                                |
| -------------- | ---------------------------------------- |
| **open**       | Service luistert actief                  |
| **closed**     | Host reageert, maar geen service         |
| **filtered**   | Firewall blokkeert probes                |
| **unfiltered** | Probe ontvangen, maar geen staat bepaald |
| **open\|filtered**   | Onvoldoende info                         |
| **closed\|filtered** | Onvoldoende info                         |

### Meerdere targets scannen

**Losse hosts:**
```bash
nmap host1 host2 host3
```

**CIDR / subnet:**
```bash
nmap 10.0.0.0/24
```

**Range:**
```bash
nmap 10.0.0.1-50
```

**Lijst in bestand:**
```bash
nmap -iL targets.txt
```

**Hosts uitsluiten:**
```bash
nmap 10.0.0.0/24 --exclude 10.0.0.5
nmap --exclude-file exclude.txt
```

### Interface forcing

```bash
nmap -e eth0 target
```

### IPv6 scannen

```bash
nmap -6 target
```

::note
Sommige Nmap features werken niet met IPv6 (ranges, CIDR notation).
::

### Willekeurige hosts scannen

```bash
nmap -iR 10          # Scan 10 random hosts
```

::warning
Gebruik dit zeer spaarzaam - kan tot ISP problemen leiden.
::

### Reason

```bash
nmap --reason target
```

```bash
PORT    STATE    REASON
22/tcp  open     syn-ack
80/tcp  closed   reset
443/tcp filtered no-response
```

::note
Unleash the power of port-state intelligence — zie exact waarom Nmap beslist wat het beslist.
::

### Alleen open poorten tonen

```bash
nmap --open target
```

### Packet trace

```bash
nmap --packet-trace target
```

Laat **elk verstuurd en ontvangen** pakket zien tijdens de scan.
