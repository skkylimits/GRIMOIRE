## 🐍 Nmap Scripting Engine (NSE)

### Script uitvoeren

```bash
nmap --script dns-brute target
nmap --script default target            # Safe default scripts
nmap --script vuln target               # Vulnerability detection
nmap --script exploit target            # Exploitation (dangerous!)
nmap --script "http-*" target           # All HTTP scripts
```

### Script Categories

| **Category** | **Beschrijving** |
|--------------|------------------|
| `auth` | Authentication bypass/bruteforce |
| `broadcast` | Network broadcast/discovery |
| `brute` | Credential bruteforcing |
| `default` | Safe, useful, fast scripts |
| `discovery` | Service/host discovery |
| `dos` | Denial of service (use carefully!) |
| `exploit` | Active exploitation |
| `fuzzer` | Fuzzing attacks |
| `intrusive` | Likely to crash target |
| `malware` | Malware detection |
| `safe` | Won't affect target |
| `version` | Advanced version detection |
| `vuln` | Vulnerability detection |

### Useful examples

**WHOIS info ophalen:**
```bash
nmap -sn --script whois* target
```

**Traceroute geolocation:**
```bash
nmap --traceroute --script traceroute-geolocation target
```

**HTTP enumeration:**
```bash
nmap --script http-enum target
```

**SMB enumeration:**
```bash
nmap --script smb-enum-shares,smb-enum-users target
```

**Vulnerability scanning:**
```bash
nmap --script vulners,vulscan target
```

### Script arguments

```bash
nmap --script http-brute --script-args http-brute.path=/admin target
```

### Script help

```bash
nmap --script-help http-enum
```

::important
NSE scripts kunnen **intrusive** zijn - gebruik alleen op authorized targets!
::
