## 📄 Output Opties

### Output formats

**Tekst, XML, Greppable:**
```bash
nmap -oN output.txt target              # Normal text
nmap -oX output.xml target              # XML format
nmap -oG output.grep target             # Greppable format
```

**Alle tegelijk:**
```bash
nmap -oA scan target                    # Creates scan.nmap, scan.xml, scan.gnmap
```

::tip
`-oA` is handig voor complete documentatie - je hebt dan alle formaten.
::

### Append to file

```bash
nmap -oN output.txt --append-output target
```

### Periodieke status

```bash
nmap --stats-every 2s target            # Stats elke 2 seconden
nmap --stats-every 10s target           # Stats elke 10 seconden
```

**During scan:** Press `SPACE` voor on-demand stats.

::note
Stats zijn handig bij lange scans om progress te monitoren zonder verbose output.
::
