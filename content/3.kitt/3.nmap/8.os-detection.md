## 🖥️ Operating System Detectie

### OS fingerprinting

```bash
nmap -O target
```

**Requirements:**
- Minimaal 1 open poort
- Minimaal 1 closed poort

### OS guess (aggressive)

```bash
nmap -O --osscan-guess target
nmap -O --fuzzy target           # Synonym
```

**Example output:**
```
Running: Linux 5.X
OS CPE: cpe:/o:linux:linux_kernel:5
OS details: Linux 5.0 - 5.4
```

::note
OS detection werkt door TCP/IP stack fingerprinting - verschillen in implementaties tussen OS'en.
::
