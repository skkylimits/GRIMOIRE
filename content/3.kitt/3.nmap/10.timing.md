## ⏱️ Timing (T0–T5)

### Timing templates

| Template | Snelheid       | Gebruik        |
| -------- | -------------- | -------------- |
| **T0**   | Zeer traag     | IDS ontduiking |
| **T1**   | Traag          | Stealth        |
| **T2**   | Normaal traag  | WAN scans      |
| **T3**   | Standaard      | Balanced       |
| **T4**   | Snel           | LAN scans      |
| **T5**   | Zeer agressief | Lab only       |

**Gebruik:**
```bash
nmap -T0 target              # Paranoid (IDS evasion)
nmap -T1 target              # Sneaky
nmap -T2 target              # Polite
nmap -T3 target              # Normal (default)
nmap -T4 target              # Aggressive (recommended voor LAN)
nmap -T5 target              # Insane (inaccurate!)
```

::tip
Voor productie gebruik: **T3** (default) of **T4** (LAN). T5 produceert vaak inaccurate results.
::

### Custom timing parameters

```bash
nmap --min-parallelism 100 target       # Min parallel scans
nmap --max-parallelism 1 target         # Serial scanning
nmap --min-rtt-timeout 100ms target     # Min response timeout
nmap --max-rtt-timeout 10s target       # Max response timeout
nmap --max-retries 3 target             # Max probe retransmissions
nmap --host-timeout 30m target          # Skip host after 30min
nmap --scan-delay 1s target             # Delay tussen probes
nmap --max-scan-delay 10s target        # Max delay cap
nmap --min-rate 100 target              # Min packets/second
nmap --max-rate 100 target              # Max packets/second
```

::note
Timing suffixes: `s` (seconds), `m` (minutes), `h` (hours). Default is milliseconds.
::
