## 🔍 Service Version Detection

### Basis version detection

```bash
nmap -sV target
```

### Aggressive version detection

```bash
nmap -sV --version-intensity 9 target    # 0-9, default 7
```

### Verbose version trace

```bash
nmap -sV --version-trace target          # Debug output
```

**Example output:**
```
PORT   STATE SERVICE VERSION
22/tcp open  ssh     OpenSSH 8.2p1 Ubuntu 4ubuntu0.5
80/tcp open  http    Apache httpd 2.4.41 ((Ubuntu))
```

::tip
Version detection probeert specifieke probes te sturen naar services om exact version info te krijgen.
::
