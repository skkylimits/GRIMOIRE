## 🛡️ Firewall Evasion & Obfuscation

### MTU aanpassen

```bash
nmap --mtu 16 target         # Must be multiple of 8
```

### Decoys (verhult jouw bron-IP)

```bash
nmap -D RND:5 target                    # 5 random decoys
nmap -D 192.168.1.10,192.168.1.20,target  # Specific decoys
```

::caution
Decoys werken niet als:
- ISP filtert spoofed traffic
- Target heeft ingress filtering
- Te veel decoys → network congestion
::

### Idle / Zombie Scan

```bash
nmap -sI zombieIP target
```

**Requirements:**
- Zombie host must be truly idle (no other traffic)
- Predictable IP ID sequence generation

::note
Idle scans sturen **geen** packets van jouw IP naar de target - zeer stealth!
::

### Source port spoofing

```bash
nmap --source-port 53 target # Spoof DNS port
nmap -g 20 target            # Synonym (FTP-data)
```

**Common bypass ports:** 20 (FTP), 53 (DNS), 67 (DHCP), 88 (Kerberos)

### Packet payload vullen

```bash
nmap --data-length 25 target # Append random data
```

### Random host order

```bash
nmap --randomize-hosts target
```

### MAC Spoofing

```bash
nmap --spoof-mac 0 target                # Random MAC
nmap --spoof-mac AA:BB:CC:DD:EE:FF target # Specific MAC
nmap --spoof-mac Apple target            # Vendor name
```

### Bad checksums

```bash
nmap --badsum target
```

::important
Goed geconfigureerde systemen **droppen** bad checksum packets. Als je response krijgt, is er een firewall/IDS die packets niet valideert.
::
