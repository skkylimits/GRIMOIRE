## 🎨 Zenmap GUI

**Zenmap** is de officiële grafische interface voor Nmap.

### Voordelen

* **Template scans** - Pre-configured scan profiles (Quick, Intense, etc.)
* **Topology mapping** - Visuele weergave van netwerken
* **Scan comparison** - Vergelijk resultaten tussen scans
* **Results export** - Exporteer naar verschillende formaten
* **Command builder** - Build complex commands visueel

### Starten

```bash
zenmap
```

Of met sudo voor privileged scans:

```bash
sudo zenmap
```

### Pre-configured profiles

| Profile | Beschrijving |
|---------|--------------|
| **Quick scan** | `nmap -T4 -F target` |
| **Quick scan plus** | `nmap -sV -T4 -O -F --version-light target` |
| **Quick traceroute** | `nmap -sn --traceroute target` |
| **Intense scan** | `nmap -T4 -A -v target` |
| **Intense scan + UDP** | `nmap -sS -sU -T4 -A -v target` |
| **Comprehensive scan** | `nmap -sS -sU -T4 -A -v -PE -PP -PS80,443 -PA3389 -PU40125 -PY -g 53 --script "default or (discovery and safe)" target` |

::tip
Zenmap is perfect voor beginners om Nmap te leren - je ziet de commando's die gegenereerd worden.
::

### Network topology view

Zenmap kan een visuele topology map maken van je netwerk:
- Hosts als nodes
- Connections tussen hosts
- OS icons
- Service info tooltips

::note
Zenmap is vooral handy voor:
- Learning Nmap syntax
- Quick visual scans
- Network documentation (screenshots)
- Comparing scan results over time
::
