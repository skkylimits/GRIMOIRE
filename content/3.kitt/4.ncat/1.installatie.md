## 📦 Installatie Op Linux

Ncat wordt geleverd met het **Nmap package**. Installatie is simpel:

### Debian/Ubuntu/Kali

```bash
sudo apt update
sudo apt install nmap -y
```

::note
De meeste Linux distributies hebben ncat **by default geïnstalleerd** of beschikbaar in repositories. Kali Linux en andere pentest distro's hebben het altijd pre-installed.
::

---

## 🪟 Installatie Op Windows

### Optie 1: Nmap Installer (Aanbevolen)

1. Download Nmap van https://nmap.org/download.html
2. Run de Windows installer (`nmap-<version>-setup.exe`)
3. Zorg dat **"Ncat"** geselecteerd is tijdens installatie
4. Voeg Nmap directory toe aan PATH (optioneel maar handig)

**Installatie directory (default):**
```
C:\Program Files (x86)\Nmap\
```

### Optie 2: Portable Executable

Download de standalone `ncat.exe` van de Nmap website:
```
https://nmap.org/dist/ncat-portable-<version>.zip
```

Unzip en run direct — geen installatie nodig.

::tip
**Voor pentesting:** Portable executable is handig om mee te nemen naar targets zonder admin rechten nodig te hebben.
::

## ✅ Verificatie

### Controleer Of Ncat Geïnstalleerd Is

```bash
ncat --version
```

**Verwachte output:**
```
Ncat: Version 7.94 ( https://nmap.org/ncat )
```

### Test Basic Functionaliteit

**Simpele echo test (twee terminals):**

Terminal 1 (listener):
```bash
ncat -l 4444
```

Terminal 2 (client):
```bash
ncat localhost 4444
```

Type tekst in een van beide terminals — het moet verschijnen in de andere.

Press `Ctrl+C` om te stoppen.

::note
Als dit werkt, is ncat correct geïnstalleerd en operationeel!
::

## 🔧 Alternative Netcat Varianten Installeren

### Traditional Netcat (nc)

**Debian/Ubuntu:**
```bash
sudo apt install netcat-traditional -y
# Of de OpenBSD variant:
sudo apt install netcat-openbsd -y
```

**Verschil:**
- `netcat-traditional` — originele implementatie met `-e` flag
- `netcat-openbsd` — moderne, veiligere versie (geen `-e`)

::caution
Veel moderne Linux distro's hebben de **OpenBSD variant** als default `nc`, die de `-e` flag **niet** ondersteunt (security redenen). Voor reverse shells met `nc -e` moet je de traditional variant installeren.
::

---

### Socat

**Linux:**
```bash
sudo apt install socat -y
```

**Windows:**
- Download van https://github.com/StudioEtrange/socat-windows

**Verificatie:**
```bash

```

### PowerCat (Windows)

PowerCat is een **PowerShell script**, geen binary. Download het:

```powershell
# Download PowerCat.ps1
IEX(New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/besimorhino/powercat/master/powercat.ps1')

# Of download lokaal
Invoke-WebRequest -Uri https://raw.githubusercontent.com/besimorhino/powercat/master/powercat.ps1 -OutFile powercat.ps1
```

**Gebruik:**
```powershell
# Import de module
Import-Module .\powercat.ps1

# Test
powercat -h
```

::warning
**Execution Policy:** Windows kan PowerShell scripts blokkeren. Bypass (alleen in lab):
```powershell
powershell -ExecutionPolicy Bypass -File powercat.ps1
```
::

## 🐧 Linux — Welke Netcat Heb Je?

Veel Linux systemen hebben **meerdere netcat varianten** geïnstalleerd. Check welke je gebruikt:

```bash
which nc
ls -la $(which nc)
```

**Mogelijk output:**
```bash
/bin/nc -> /etc/alternatives/nc -> /bin/nc.openbsd
```

Dit toont dat `nc` een symlink is naar de OpenBSD variant.

### Switch Tussen Varianten (Debian/Ubuntu)

```bash
# Toon beschikbare alternatieven
sudo update-alternatives --config nc

# Output:
#   Selection    Path                     Priority   Status
# ------------------------------------------------------------
# * 0            /bin/nc.openbsd          50        auto mode
#   1            /bin/nc.openbsd          50        manual mode
#   2            /bin/nc.traditional      10        manual mode

# Kies '2' voor netcat-traditional (met -e flag)
```

::important
**Voor pentest work:** Als je `nc -e` nodig hebt, zorg dat je de **traditional variant** hebt geïnstalleerd en geselecteerd.
::
