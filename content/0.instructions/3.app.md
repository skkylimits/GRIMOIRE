## New Directory Structure

This is a new way of structuring ones app to bundle previously loose directories like `component/` and `layers/` into a unified `app/` directory.

This aligns with Nuxt v4’s streamlined architecture, making the codebase more organized, efficient, and future-ready for the v4 compatibility transition.

::callout
---
icon: i-mdi:nuxt
target: _blank
to: https://nuxt.com/docs/getting-started/upgrade#new-directory-structure
---
[Learn more about the `new directory structure`]()
::

## Example Directory Structure

```
.output/
.nuxt/
app/
  assets/
  components/
  composables/
  layouts/
  middleware/
  pages/
  plugins/
  utils/
  app.config.ts
  app.vue
  router.options.ts
content/
layers/
modules/
node_modules/
public/
server/
  api/
  middleware/
  plugins/
  routes/
  utils/
nuxt.config.ts
```
