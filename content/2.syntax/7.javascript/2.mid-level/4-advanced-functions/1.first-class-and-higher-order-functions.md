---
title: First Class and Higher Order Functions
---

Functions in JavaScript are treated much like any other variable. This means they can be passed around as the argument to other functions or even returned from functions. Learn how to take advantage of the first class nature of functions to create higher order functions.

- MDN > Callback function - https://developer.mozilla.org/en-US/docs/Glossary/Callback_function
