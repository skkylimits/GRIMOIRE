## 1 Which method is used to select a single element by its ID in JavaScript?

document.getElementsByClassName('myElement');

**document.getElementById('myElement');**

document.getElementByID('myElement');

## 2 Which property is used to access the parent node of an element in the DOM?

element.sibling;

element.child;

element.ancestor;

**element.parentNode;**

## 3 Which method is used to create a new element in the DOM?

document.generateElement('div');

document.newElement('div');

**document.createElement('div');**

document.makeElement('div');

## 4 Which property is used to change the text content of an element?

element.text = 'New content';

element.content = 'New content';

element.innerHTML = 'New content';

**element.textContent = 'New content';**

## 5 Which is the correct way to change the background color of an element using JavaScript?

element.css('backgroundColor', 'blue');

**element.style.backgroundColor = 'blue';**

element.style.background-color = 'blue';

element.backgroundColor = 'blue';
