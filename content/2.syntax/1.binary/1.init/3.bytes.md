## Wat is een byte?

* Een `byte` bestaat uit **8 bits**.
* Eén byte kan `2^8 = 256` waarden voorstellen (`0–255`).
* Bytes vormen de standaard-eenheid voor dataopslag en geheugen.

```text
1 byte = 8 bits

b7 b6 b5 b4 b3 b2 b1 b0
 |  |  |  |  |  |  |  |
 1  0  0  1  0  1  1  0
```

**Voorbeeld:**
`01000001` = **65** (decimaal) = `'A'` (ASCII).

```text
01000001₂ = 64 + 1 = 65₁₀ → 'A'
```

## **Waarom 8 bits?**

In de beginjaren van computers (jaren ’50–’60) bestond er geen vaste bytegrootte.  
Elke fabrikant koos zijn eigen formaat — 6, 7, 9, 12 of zelfs 36 bits per `woord`.

| Computer         | Jaar | Bytegrootte             |
| ---------------- | ---- | ----------------------- |
| IBM 7030 Stretch | 1961 | 8 bits                  |
| DEC PDP-10       | 1966 | 7 bits                  |
| CDC 6600         | 1964 | 10×6-bit woorden        |
| UNIVAC 1108      | 1964 | 36 bits                 |

Toen in 1963 het `ASCII`-tekensysteem werd ontworpen, had het `7 bits` nodig voor 128 tekens.  
Om een extra `pariteitsbit` (voor foutdetectie) toe te voegen én de hardware mooi op een `macht van twee` te houden,  
werd de `8-bit byte` ingevoerd.

```

2⁸ = 256 mogelijke waarden (0–255)

```

Dat bleek ideaal:
- genoeg ruimte voor letters, cijfers en controle-symbolen;  
- één extra bit voor foutdetectie;  
- efficiënt te verwerken op binaire hardware (`2ⁿ`-structuur).

Toen IBM en later Intel hun architecturen standaardiseerden op `8-bit bytes`,  
werd dit wereldwijd de norm —  
en groeide de byte uit tot de `fundamentele bouwsteen` van alle moderne computers.

::note
**Kort gezegd:** 

-8 bits was precies de juiste balans tussen eenvoud, snelheid en compatibiliteit.

## Grenzen van 8 bits

Hoewel `8 bits` lang de standaard was, raakt die limiet steeds sneller bereikt.
Net als bij **ASCII** (te weinig tekens) en **IPv4** (te weinig adressen) zien we dat vaste 8-bit structuren beperkt zijn.

In latere hoofdstukken (zoals `Encoding`, `Networking` en `System Engineering`) duiken we dieper in hoe systemen dit oplossen met grotere formaten zoals **UTF-8** en **IPv6**.
