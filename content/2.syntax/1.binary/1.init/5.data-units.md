> “Before we can encode information, we must first measure it.”

Computers leven in een wereld van `bits`, `bytes`, en `woorden` —
maar zodra je data gaat opslaan of verzenden, komt er iets bij: `eenheden van grootte`.

En daar ontstaat vaak verwarring:
🧮 is 1 kilobyte nou 1000 bytes of 1024 bytes?

Laten we dat één keer goed helder maken.

## 🧠 Bits en Bytes

- `1 bit` → kleinste eenheid van informatie, 0 of 1
- `1 byte` → groep van 8 bits

```
8 bits  = 1 byte
1024 bytes = 1 KiB
```

De byte werd de standaard omdat computers meestal in blokken van 8 bits rekenen
(dit past bij CPU-registers, geheugenadressering en bestandsstructuren).

## 📏 Twee meetsystemen: SI vs Binary

Er bestaan twee “talen” om datagroottes uit te drukken:

| Systeem | Gebruikt door | Basis | Vermenigvuldiger |
|----------|----------------|--------|------------------|
| `SI (decimaal)` | opslagfabrikanten, marketing | 10 | `1 kB = 1000 bytes` |
| `Binaire (IEC)` | besturingssystemen, tech | 2 | `1 KiB = 1024 bytes` |

## 📚 De tabel van waarheid

| Eenheid | Symbool | Vermenigvuldiger | Bytes | Bits |
|----------|----------|------------------|--------|------|
| `Bit` | `b` | – | 1/8 byte | 1 |
| `Byte` | `B` | 8 bits | 1 | 8 |
| `Kilobyte` | `kB` | 10³ | 1,000 | 8,000 |
| `Kibibyte` | `KiB` | 2¹⁰ | 1,024 | 8,192 |
| `Megabyte` | `MB` | 10⁶ | 1,000,000 | 8,000,000 |
| `Mebibyte` | `MiB` | 2²⁰ | 1,048,576 | 8,388,608 |
| `Gigabyte` | `GB` | 10⁹ | 1,000,000,000 | 8,000,000,000 |
| `Gibibyte` | `GiB` | 2³⁰ | 1,073,741,824 | 8,589,934,592 |

## 🧮 Voorbeeld: de “verloren megabytes”

Je koopt een `1 TB`-schijf.
De fabrikant rekent in SI:

```
1 TB = 1000^4 bytes = 1,000,000,000,000 bytes
```

Maar jouw besturingssysteem (bijv. Windows of Linux) rekent in binaire stappen:

```
1 TiB = 1024^4 bytes = 1,099,511,627,776 bytes
```

**➜ Resultaat:**
```text
1,000,000,000,000 / 1,099,511,627,776 = 0.91 TiB
```

Je ziet dus ~931 “GB” in plaats van 1 TB.
Geen data verdwenen — alleen verschillende rekenstelsels.

::warning
Bits vs Bytes in netwerksnelheid

- Let op: internetproviders meten in `bits per seconde (b/s)`,
maar besturingssystemen tonen `bytes per seconde (B/s)`.

::

| Eenheid | Uitleg                                           |
| ------- | ------------------------------------------------ |
| `Mbps`  | megabit per seconde (bandbreedte)                |
| `MB/s`  | megabyte per seconde (werkelijke dataoverdracht) |

Omrekenen:

```
1 MB/s = 8 Mbps
```

Dus een `100 Mbps` verbinding → max. ±12.5 MB/s effectief.


## 🧩 Waarom dit ertoe doet

* In `reverse engineering` en `forensics` moet je exact weten hoeveel bytes iets gebruikt.
* In `networking` en `malware-analysis` moet je bits ↔ bytes correct kunnen omrekenen.
* In `storage security` bepaalt het of je juiste schijfgroottes, buffers en payload-limieten hanteert.

> ❗️Elke byte telt — vooral als die ene byte bepaalt of je buffer overloopt.


## ⚡ TDL;DR

| Concept           | Sleutelgedachte                 |
| ----------------- | ------------------------------- |
| `1 bit`           | kleinste informatie-eenheid     |
| `1 byte`          | 8 bits                          |
| `SI vs Binary`    | 1000 vs 1024                    |
| `Fabrikanten`     | gebruiken SI (grotere getallen) |
| `OS / Tech`       | gebruikt binaire waarden        |
| `1 MB/s = 8 Mbps` | snelheid ≠ opslagmaat           |

**Kort gezegd:**

> Data-units zijn de taal waarin opslag, geheugen en netwerken met elkaar praten.
> Begrijp je de eenheden, dan begrijp je hoeveel informatie je écht hebt — of mist.
