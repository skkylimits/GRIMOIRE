> From Logic to Information

## Wat is een Bit

Een `bit` (afkorting van *binary digit*) is de kleinste eenheid van informatie in een computer.
Een bit heeft maar twee mogelijke waarden: `0` of `1`.

Deze twee toestanden corresponderen direct met de fysieke werkelijkheid die je net hebt leren kennen:
`0` = geen stroom, `1` = stroom aanwezig.

```text
  ┌───────┐
  │ 0  → uit │
  │ 1  → aan │
  └───────┘
```

Elke bit is dus een kleine *ja/nee*, *waar/onwaar*, of *aan/uit* beslissing.

## Bits in combinatie

Één bit kan slechts twee toestanden weergeven,
maar met meerdere bits kun je exponentieel meer informatie voorstellen:

```text
Aantal bits │ Mogelijke combinaties
────────────┼──────────────────────
     1      │ 2^1 = 2
     2      │ 2^2 = 4
     3      │ 2^3 = 8
     4      │ 2^4 = 16
     8      │ 2^8 = 256
```

Voorbeeld:
Met 3 bits kun je waarden `000` tot `111` weergeven — acht verschillende toestanden.

```text
000 → 0
001 → 1
010 → 2
011 → 3
100 → 4
101 → 5
110 → 6
111 → 7
```

## Bits in actie

Computers gebruiken bits overal:

* In geheugen (`RAM`) — elke cel slaat een bit op (geladen of ontladen).
* In opslag (`SSD`/`HDD`) — magnetische of elektrische lading per bit.
* In netwerkverkeer — bits bewegen als signalen over kabels of radiofrequenties.

Elk bestand, elke afbeelding, elk programma is in essentie een lange keten van bits:

```text
01001000 01100101 01101100 01101100 01101111
```

⬆️ Dat is het woord **HELLO** in binaire vorm (ASCII).

## Visualisatie: een bit in geheugen

```text
+---------+
| Charge  | → 1  (aanwezig)
+---------+

+---------+
| No charge | → 0  (afwezig)
+---------+
```

Bits zijn dus als kleine containers van spanning, elk met één simpele boodschap:

> “Ik ben aan” of “Ik ben uit.”

## Waarom bits belangrijk zijn

Alle vormen van data — getallen, tekst, kleur, geluid — worden in bits gecodeerd.
Zelfs complexe concepten zoals encryptie, hashing en netwerkpakketten beginnen met dezelfde simpele binaire bouwstenen.

Wie bits begrijpt, begrijpt hoe computers *denken*.
