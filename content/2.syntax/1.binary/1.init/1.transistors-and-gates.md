## Transistors

Elke computer begint bij `stroom` вҖ” letterlijk.
Elektriciteit kan `aan` of `uit` staan, en dat simpele verschil vormt de basis van alles wat digitaal is.

Een `transistor` is het kleine onderdeel dat deze twee toestanden bestuurt:

* wanneer hij `stroom doorlaat` вҶ’ dat noemen we een `1` (aan)
* wanneer hij `stroom blokkeert` вҶ’ dat noemen we een `0` (uit)

### рҹ”Ң Transistor als schakelaar

Wanneer de transistor **open** is, vloeit er stroom вҖ” de uitgang is `1`.
Wanneer hij **gesloten** is, wordt de stroom geblokkeerd вҖ” de uitgang is `0`.

```text
     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
IN в”Җв–¶в”Ӯ  Transistor  в”Ӯв”Җв”Җв”Җв–¶ OUT
     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
          в”Ӯ
          в–ј
      Control (Base)

```

Wanneer de `control` (basis) spanning krijgt, вҖңsluitвҖқ de transistor het circuit en stroomt er stroom door.
Geen spanning = geen stroom вҶ’ `0`
Wel spanning = stroom vloeit вҶ’ `1`

## Logic Gates

Door meerdere `transistoren` te combineren ontstaan **`logic gates`** вҖ” kleine schakelingen die beslissingen nemen op basis van `1` en `0`.
Elke gate volgt een eenvoudige logische regel: afhankelijk van de ingangen bepaalt ze wat de uitgang wordt.

De drie basisvormen zijn `AND`, `OR` en `NOT`.
Alle andere logische functies kunnen uit deze drie worden opgebouwd.

---

### рҹ”№ AND-gate

Geeft `1` **alleen** als *beide* ingangen `1` zijn.
In elk ander geval is de uitgang `0`.

```text
 A   B  в”Ӯ  Output
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”јв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
 0   0  в”Ӯ   0
 0   1  в”Ӯ   0
 1   0  в”Ӯ   0
 1   1  в”Ӯ   1
```

**Visualisatie:**

```text
   A в”Җв”Җв”Җв”җ
        в”Ӯ
        в”ңв”Җв”Җв–¶ (AND) в”Җв”Җв–¶ OUTPUT
        в”Ӯ
   B в”Җв”Җв”Җв”ҳ

Voorbeeld:
Twee schakelaars in serie вҖ”
beide moeten aan staan om stroom door te laten.
```

---

### рҹ”№ OR-gate

Geeft `1` als **minstens Г©Г©n** ingang `1` is.
Alleen `0` als *beide* ingangen `0` zijn.

```text
 A   B  в”Ӯ  Output
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”јв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
 0   0  в”Ӯ   0
 0   1  в”Ӯ   1
 1   0  в”Ӯ   1
 1   1  в”Ӯ   1
```

**Visualisatie:**

```text
   A в”Җв”Җв”Җв”җ
        в”ңв”Җв”Җв–¶ (OR) в”Җв”Җв–¶ OUTPUT
   B в”Җв”Җв”Җв”ҳ

Voorbeeld:
Twee schakelaars parallel вҖ”
Г©Г©n van de twee hoeft aan te staan om stroom door te laten.
```

### рҹ”№ NOT-gate

Keert de waarde om: `1` wordt `0`, `0` wordt `1`.
Wordt ook wel een **inverter** genoemd.

```text
 Input в”Ӯ Output
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”јв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
   0   в”Ӯ   1
   1   в”Ӯ   0
```

**Visualisatie:**

```text
   INPUT в”Җв”Җв–¶[ NOT ]в”Җв”Җв–¶ OUTPUT
                  вҶ‘
        Kleine cirkel (вҖў) = omkering

Voorbeeld:
Als het licht aan is вҶ’ NOT maakt het uit.
Als het uit is вҶ’ NOT maakt het aan.
```

---

Deze drie bouwstenen vormen de kern van digitale logica.
Met combinaties van `AND`, `OR` en `NOT` kun je alles bouwen вҖ”
van eenvoudige condities tot complete processoren.
