## Wat is een byte?

* Een `byte` bestaat uit **8 bits**.
* Eén byte kan `2^8 = 256` waarden voorstellen (`0–255`).
* Bytes vormen de standaard-eenheid voor dataopslag en geheugen.

```text
1 byte = 8 bits

b7 b6 b5 b4 b3 b2 b1 b0
 |  |  |  |  |  |  |  |
 1  0  0  1  0  1  1  0
```

**Voorbeeld:**
`01000001` = **65** (decimaal) = `'A'` (ASCII).

```text
01000001₂ = 64 + 1 = 65₁₀ → 'A'
```

## **Waarom 8 bits?**

* In de jaren ’60 gebruikten verschillende systemen 6, 7 of 9 bits per karakter.
* Toen **ASCII** (tekencodering) werd gestandaardiseerd, had het **7 bits** nodig → computers maakten er **8 bits** van om handig met één extra controlebit (of padding) te werken.
* 8 past ook perfect in binaire vermenigvuldiging (`2³ = 8`), wat CPU’s en geheugenontwerpen simpeler maakt.

**Kort gezegd:**

> 8 bits werd de sweet spot tussen efficiëntie, eenvoud en compatibiliteit.

## Grenzen van 8 bits

Hoewel `8 bits` lang de standaard was, raakt die limiet steeds sneller bereikt.
Net als bij **ASCII** (te weinig tekens) en **IPv4** (te weinig adressen) zien we dat vaste 8-bit structuren beperkt zijn.

In latere hoofdstukken (zoals `Encoding`, `Networking` en `System Engineering`) duiken we dieper in hoe systemen dit oplossen met grotere formaten zoals **UTF-8** en **IPv6**.
