## 🧩 1️⃣ Wat zijn Classful Networks?

In de begindagen van het internet (jaren ’80 – vóór CIDR)
werden IP-adressen verdeeld in vaste *klassen* — gebaseerd op de **eerste bits van het adres**.

Elke klasse had een vaste verhouding tussen:

* netwerkbits
* hostbits
* standaard subnetmask

---

## 🧱 2️⃣ Overzicht van de 5 IPv4-klassen

| Klasse | Eerste bits | Adresbereik                 | Default mask  | # Netwerkbits | # Hostbits | Adressen per netwerk | Gebruik                      |
| :----- | :---------- | :-------------------------- | :------------ | :-----------: | :--------: | :------------------: | :--------------------------- |
| **A**  | `0xxxxxxx`  | 1.0.0.0 – 126.255.255.255   | 255.0.0.0     |       8       |     24     |      ~16 miljoen     | Grote organisaties           |
| **B**  | `10xxxxxx`  | 128.0.0.0 – 191.255.255.255 | 255.255.0.0   |       16      |     16     |        65.534        | Middelgrote netwerken        |
| **C**  | `110xxxxx`  | 192.0.0.0 – 223.255.255.255 | 255.255.255.0 |       24      |      8     |          254         | Kleine netwerken             |
| **D**  | `1110xxxx`  | 224.0.0.0 – 239.255.255.255 | n.v.t.        |       -       |      -     |           -          | Multicast                    |
| **E**  | `1111xxxx`  | 240.0.0.0 – 255.255.255.255 | n.v.t.        |       -       |      -     |           -          | Experimenteel (gereserveerd) |

## 🧠 4️⃣ Waarom dit systeem slecht schaalde

Classful addressing werkte goed in het begin, maar had **enorme verspilling**:

* Een Class A netwerk gaf je 16 miljoen adressen, zelfs als je er maar 200 nodig had.
* Een Class C had er 254 — te weinig voor veel organisaties.
* Geen middenweg → **inefficiëntie en IP-tekort**.

---

## ⚙️ 5️⃣ De overgang naar CIDR (1993)

Toen kwam **CIDR (Classless Inter-Domain Routing)**:
in plaats van die vaste klassen, konden netwerken nu flexibel zijn — elk willekeurig aantal netwerkbits.

Bijv.:

```
Class C → altijd /24 → 254 hosts
CIDR   → /26 → 62 hosts
         /28 → 14 hosts
         /30 → 2 hosts
```

> 💡 CIDR **vervanging van klassen**, maar de *range* van A/B/C is nog steeds handig
> om snel te herkennen in welk type IP-bereik je zit.

---

## 📡 6️⃣ Samenvattend verschil

| Eigenschap      | Classful              | CIDR          |
| --------------- | --------------------- | ------------- |
| Periode         | 1981–1993             | 1993–nu       |
| Netwerkgrenzen  | Vast (A/B/C)          | Variabel (/n) |
| Subnetmask      | Standaard per klasse  | Flexibel      |
| Routing         | Class-based           | Prefix-based  |
| Gebruik vandaag | Alleen nog historisch | Overal actief |
