## Do Experienced Linux users remember all the commands?

When I was first learning about Linux as a total noob. I asked a similar question of a far-more-experienced friend of mine. They responded by opening up a terminal on a Linux machine and running the following command

```bash [terminal]
ls /usr/bin | wc -l
```

It should output a number indicating the number of prgrams available in /usr/bin, which is where user-installed software generally goes, and should give us a good estimate for how many commands are available for a user to run from the terminal.

On my laptop, that number is `3468` -- it wil vary from machine to machine, but will likely be of similar magnitude unless you're running a very minimal distro. And that's just user-installed software -- there will also be a (smaller) number of executables available in /bin and perhaps user/local/bin.

While it's conceivable possible to memorize all of these, I doubt most users -- even experts who use Linux every day -- would bother
