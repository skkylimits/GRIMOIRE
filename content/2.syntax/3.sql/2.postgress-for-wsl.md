## Installation 

https://www.postgresql.org/download/linux/debian/

```bash
sudo sh -c 'echo "deb https://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list'

wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add 
-
sudo apt-get update

sudo apt-get -y install postgresql
```

## Start Server

```bash
sudo service postgresql start
```

## CREATE DB

```bash
CREATE ROLE yourUser PASSWORD 'yourpass' SUPERUSER CREATEDB CREATEROLE INHERIT LOGIN;
```

## Install PGadmin4

Dowload https://www.pgadmin.org/download/pgadmin-4-windows/

## CONNECT PG4 WITH PSQL

Ensure the connection settings in pgAdmin are correct:

- **Host name/address:** Use localhost 127.0.0.1
- **Port:** The default PostgreSQL port is 5432.
- **Maintenance database:** Usually set to "postgres."
- **Username/Password:** The username and password used when creating the db.
