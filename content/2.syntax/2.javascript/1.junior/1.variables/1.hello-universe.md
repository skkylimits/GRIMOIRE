## Baby Steps

::steps{level=4}
#### Download the project using the button at the bottom of this page.

#### Unzip the project.

#### Open the project in your IDE, startup the development server, and visit the page in the browser (probably http://localhost:5173/)

::tip
If you need help starting up the development server and running this challenge in the browser see the lesson [`How to Startup the Coding Challenges`](/syntax/javascript/prerequisites/challenges)
::

#### Open the file src/main.js

#### Add the following code to the file:

```js
alert('Hello Universe')
```

#### What happened? You should see an alert in the browser that says "Hello Universe". You can click "Ok" to dismiss it.

![alert](/content/2.syntax/2.javascript/1.variables/1.hello-universe/alert.png)

#### Replace your alert code with the following:

```js
console.log('Hello console')
```

#### In the browser, nothing appears to happen but our message is just hiding.

#### To see the message open the browser developer console by right clicking on the page and selecting "Inspect" or pressing `Q`

![inspect](/content/2.syntax/2.javascript/1.variables/1.hello-universe/inspect.png)

::tip
Press F12 for fast access to the console.
::

#### This opens the browser Developer Tools

#### Click on the "Console" Tab and you will see the message!

![console](/content/2.syntax/2.javascript/1.variables/1.hello-universe/console.png)

#### Congrats! You've written your first bit of JavaScript! 🎉
::
