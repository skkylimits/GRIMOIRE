## Fix the Syntax Error And Log a Custom Message to the Console

In this challenge you'll need to fix the syntax error found in /src/main.js and then call the provided sum function to add 2 + 5. Finally you should console log the answer.

## Requirements

- Fix the syntax error in /src/main.js

![remove-syntax-error](/content/2.syntax/2.javascript/mid-level/1-fundamentals/3-challenge-1/remove-syntax-error.jpg)

- Call the provided sum function, adding 2 + 5
- Log the answer (7) to the browser console

![console-log](/content/2.syntax/2.javascript/mid-level/1-fundamentals/3-challenge-1/console-log.jpg)

::tip
you could also call alert() and pass it the answer like this alert(answer). Alert used to be a useful for communicating feedback to users. Most web applications now use more nicely styled custom solutions but alert can still be useful for debugging just like console log.
::
