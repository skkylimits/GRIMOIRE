## 1 What is a closure in JavaScript?

A technique to prevent functions from being called more than once

A method to close unused functions and free up memory

A way to lock variables so they can't be changed

**A function that has access to variables in its outer (enclosing) lexical scope**

## 2 Which of the following best describes a callback function?

A function that returns immediately without doing any work

A function that interrupts the main program flow

A function that calls itself recursively

**A function passed as an argument to another function and executed later**

## 3 Which feature of JavaScript demonstrates that functions are first-class citizens?

**Functions can be assigned to variables, passed as arguments, and returned from other functions**

## 4 What does IIFE stand for and what is its primary purpose?

Integrated Functional Frontend: To combine HTML and JavaScript

Iterative Function For Each: To loop through arrays more efficiently

**Immediately Invoked Function Expression: To create a private scope and avoid polluting the global namespace**

Inline Internal Function Execution: To optimize function performance
