## Power Of Words

## Challenge

::steps{level="4"}

#### 1. Create a variable called `favoriteQuote` and set it to your favorite quote

1. Log the quote to the console
2. Log the length of the quote using a string method

#### 2. Create a variable called `shoutedQuote` that contains the quote in all uppercase

1. Log shoutedQuote to the console

#### 3. Create a variable called `whisperQuote` that contains the `favoriteQuote` in all lowercase letters

1. Log whisperQuote to the console
2. Log a boolean value indicating whether `whisperQuote` starts with "to" using the appropriate string metho

#### 4. Create a variable called `replacedQuote` that replaces all occurrences of "be" in `favoriteQuote` with "code"

1. Log `replacedQuote` to the console
2. Log the index of the word "code" in `replacedQuote` using the appropriate string method

#### 5. Create a variable called `trimmedQuote` that removes whitespace from favoriteQuote

1. Log trimmedQuote to the console
2. Add trailing space to favoriteQuote
3. Log trimmedQuote again to see if it changed

::

::foldable
#title
Answer

#content
```plaintext
To be or not to be, that is the question. // favoriteQuote
41 // favoriteQuote.length
TO BE OR NOT TO BE, THAT IS THE QUESTION. // shoutedQuote
to be or not to be, that is the question. // whisperQuote
true // whisperQuote.startsWith('to')
To code or not to code, that is the question. // replacedQuote
3 // replacedQuote.indexOf('code')
To be or not to be, that is the question. // trimmedQuote
To be or not to be, that is the question. // trimmedQuote after adding intentional trailing spac
```
