---
title: Authoritative DNS Server
description: An authoritative DNS server is a DNS server that stores the DNS records for a domain.
---

::important
Elke DNS-zone moet minstens twee name servers hebben die de DNS-records voor die zone beheren.
::

Meestal zijn er tussen de `twee` en `vier` nameservers voor een zone. Deze servers worden ["authoritative nameservers"](https://www.nslookup.io/learning/recursive-vs-authoritative-dns/) genoemd, omdat ze de autoriteit hebben over de DNS-records voor de domeinnamen in die zone. Alleen deze servers worden vertrouwd om de juiste DNS-records voor die zone te leveren.

![authoritative-dns-server](/content/7.knowledge-base/4.networking/1.dns/1.first-principle/5.authoratative-dns-server/authoritative-dns-server.png)

We hebben gezien dat de locaties van deze servers worden aangekondigd via NS-records. Deze records worden opgeslagen bij de nameserver van de bovenliggende zone, dus de zone die de controle heeft over de onderliggende zone. Maar ze zouden ook opgeslagen moeten worden bij de name server van de onderliggende zone.

> Bijvoorbeeld, de nameserver van de uk-zone moet zowel opgeslagen zijn bij de nameservers van de root-zone als bij de nameservers van de uk-zone.
