# 🌐 TCP Protocol Mastery

Complete, hands-on gids om TCP te begrijpen op diep technisch niveau. Alles wat je leert is **live verifieerbaar in Wireshark** met real-world packet captures.

::note
**Mission:** Na deze module kun je live TCP packets analyseren op expert-level en sta je **boven 90% van network engineers**.
::

## 🎯 Waarom TCP Masteren?

### Critical voor Security & Malware Analysis

- **90% van malware** gebruikt TCP (C2, exfiltration, downloaders)
- **Performance issues** zitten meestal in TCP, niet in netwerk hardware
- **Application delays** vs network delays onderscheiden
- **Retransmissions** correct interpreteren

### Niemand "Bezit" de Transport Layer

- Network teams kijken naar Layer 1-3
- Application teams kijken naar Layer 7
- **TCP (Layer 4)** valt ertussen → jouw expertise domein!

## 🔥 Quick Start

**Snelle introductie:**
```bash
# Open sample PCAP in Wireshark
wireshark sample-tcp.pcap

# Filter op TCP handshake
tcp.flags.syn == 1

# Bekijk sequence numbers
tcp.stream == 0
```

**Kolommen toevoegen:**
1. Right-click packet → Expand TCP
2. Find `tcp.seq`, `tcp.ack`, `tcp.len`
3. Right-click → Apply as Column
