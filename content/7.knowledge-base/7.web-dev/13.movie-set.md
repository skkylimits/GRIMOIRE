## `style.css`

> Global stylesheet to set up base styles, variables, fonts, and utilities across the app.

Responsible for:

* Importing modular CSS files like `intro.css`, `menu.css`, `contact.css`, etc.
* Setting up global variables (`--primary`, `--text`, etc.)
* Defining font faces (`BlenderProBold`, `BlenderProBook`)
* Providing base styles for HTML elements (`h1`, `a`, `section`, etc.)
* Custom cursors, hover effects, utility classes, and media queries

Useful when adding additional stylesheets for new features, layouts, or page sectionsвАФeverything stays modular but shares a consistent foundation.

## `index.js`

```js
import Experience from './Experience/Experience'
import './style.css'

const experience = new Experience(document.querySelector('canvas.webgl'))
```

* **Loads global styles**
* **Initializes** the main `Experience` class querying a `<canvas>` element** in your HTML with class `.webgl`
