## Wat is Obfuscatie?

Obfuscatie is een set technieken die de code **complexer en moeilijker te analyseren** maken, terwijl de functionaliteit behouden blijft.

**Voordelen:**

1. **Bescherming van intellectueel eigendom** – voorkomt dat code wordt gestolen of gekopieerd.
2. **Beveiliging tegen reverse engineering** – bemoeilijkt het analyseren van de logica van de code.
3. **Mitigatie van statische analyse** – bemoeilijkt automatische analyse door tools zoals antivirussoftware.

::note
Kortom, obfuscatie is een cruciaal hulpmiddel voor softwarebeveiliging en malwareontwikkeling.
::

## Voorbereiding

### 1. Visual Studio Installeren

* Download en installeer **Visual Studio 2022** (bij voorkeur).
* Voeg tijdens installatie toe:

  * **Desktop development with C++**
  * Individuele componenten:

    * Clang Compiler for Window
    * MSBuild support for LLVM (clang-cl) toolset

![ollvm-setup](/content/7.knowledge-base/7.dead-silence/3.static-detection/1.ollvm/ollvm-setup.png)

### 2. LLVM Integreren

![ollvm-binaries](/content/7.knowledge-base/7.dead-silence/3.static-detection/1.ollvm/ollvm-binaries.png)

* Download de precompiled **OLLVM binaries** (bijv. versie 13.0.1).
* Kopieer de binaries `D:\KRAIT\Windows Malware\OLLVM\ollvm-13.0.1` en overschrijf naar:

  ```
  C:\Program Files\Microsoft Visual Studio\2022\Community\VC\Tools\MSVC\1160\x64\bin
  ```

::tip
Het liefste kopieer je de binaries niet, maar laat je Visual Studio's de map inlezen, waar de binaries zich bevinden. Onder VC ++ Directories -> General -> Executable Directories.
::

![vc++-directories](/content/7.knowledge-base/7.dead-silence/3.static-detection/1.ollvm/vc++-directories.png)

### 3. Projectconfiguratie

* Zorg dat het project **pure C** is.

  * Verwijder C++-specifieke headers (`<iostream>`) en vervang door `<stdio.h>`.
  * Vermijd C++-features: classes, objects, namespaces.

## Obfuscatie Instellen in Visual Studio

### 1. Platform Toolset

* Open je project → rechterklik → **Properties** → General → **Platform Toolset** → `LLVM (clang-cl)`

![platform-toolset](/content/7.knowledge-base/7.dead-silence/3.static-detection/1.ollvm/platform-toolset.png)

### 2. Runtime Library

* Ga naar **C/C++ → Code Generation** → Runtime Library → `Multi-threaded (/MT)`
* Zorg dat de C-runtime statisch wordt gelinkt om fouten op andere machines te voorkomen.

![code-generation](/content/7.knowledge-base/7.dead-silence/3.static-detection/1.ollvm/code-generation.png)

### 3. Additional Command Line Options

Hier voeg je de obfuscatieflags toe:

**Basis obfuscatie:**

```
-Xclang -flegacy-pass-manager -Xclang -mllvm -sub -Xclang -mllvm -split -Xclang -mllvm -fla -Xclang -mllvm -bcf
```

**Uitleg flags:**

* `-sub` → instruction substitution (vervangen simpele instructies door complexe equivalenten)
* `-split` → control flow splitting (verdeelt basic blocks)
* `-fla` → control flow flattening (state machine structuur)
* `-bcf` → bogus control flow (fake conditionals)

**Advanced / heavy obfuscation:**

```
-Xclang -flegacy-pass-manager -Xclang -mllvm -sub -Xclang -mllvm -sub_loop=3 -Xclang -mllvm -split -Xclang -mllvm -split_num=3 -Xclang -mllvm -fla -Xclang -mllvm -bcf -Xclang -mllvm -bcf_loop=3 -Xclang -mllvm -bcf_prob=100
```

* `sub_loop=3` → herhaalt instruction substitution 3x
* `split_num=3` → split elk basic block in 3
* `bcf_prob=100` → 100% van alle conditionals worden vervangen door bogus logic
* `bcf_loop=3` → herhaalt bogus control flow 3x

![obfuscation](/content/7.knowledge-base/7.dead-silence/3.static-detection/1.ollvm/obfuscation.png)

## Compilatie Stappen

1. Open het project in Visual Studio
2. Stel **VC++ Directories** in
3. Stel **Platform Toolset** in op LLVM / clang4
4. Stel **Runtime Library** in op Multi-threaded (/MT)
5. Voeg obfuscatieflags toe in **C/C++ → Command Line → Additional Options**
6. Build het project **(Ctrl + B)**

::caution
Begin altijd met basis obfuscatie en verhoog complexity stapsgewijs.
* Voor API calls: gebruik dynamische resolutie / API hashing om detectie verder te bemoeilijken.
::
