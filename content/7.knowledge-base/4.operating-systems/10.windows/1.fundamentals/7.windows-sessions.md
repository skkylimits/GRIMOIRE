## 🔐 Windows Session Types

Windows onderscheidt tussen verschillende session types op basis van hoe gebruikers en processen zich authenticeren en toegang krijgen tot het systeem. Het begrijpen van deze session types is cruciaal voor zowel system administratie als security assessments.

## 👤 Interactive Sessions

Een **interactive** (of local) **logon session** wordt geïnitieerd wanneer een user zich authenticeert op een lokaal systeem of domain door credentials in te voeren.

### Interactive Logon Methoden

Interactive logon kan op verschillende manieren worden gestart:

| Methode | Beschrijving | Context |
|---------|--------------|---------|
| **Direct Login** | User logt rechtstreeks in op het systeem via keyboard/monitor | Fysieke toegang tot machine |
| **RunAs Command** | Secondary logon session via `runas` command line utility | Elevation of privileges zonder logout |
| **Remote Desktop (RDP)** | Remote interactive session via Remote Desktop Protocol | Remote toegang met GUI |
| **Fast User Switching** | Wissel tussen user sessions zonder logout | Multi-user desktop scenario |

### Interactive Session Kenmerken

::important
**Interactive Session Properties:**
- Vereist **user credentials** (username + password, smartcard, biometrics)
- Creëert een **volledig user environment** (desktop, user profile, registry hive)
- Genereert **logon events** in Security Event Log (Event ID 4624, Logon Type 2 voor local, Type 10 voor RDP)
- User kan direct **interacteren met applicaties en GUI**
- Session blijft actief totdat user uitlogt of session wordt afgebroken
::

### RunAs Secondary Logon

Het `runas` commando maakt het mogelijk om een programma te draaien onder een andere user context zonder volledig uit te loggen.

**Voorbeeld:**
```cmd
runas /user:DOMAIN\Administrator cmd.exe
runas /netonly /user:DOMAIN\ServiceAccount \\server\share
```

::tip
**RunAs Use Cases:**
- **Privilege Elevation** — Voer administrative tasks uit zonder admin account volledig te gebruiken
- **Credential Testing** — Test of credentials geldig zijn zonder in te loggen
- **Lateral Movement** — Gebruik `/netonly` flag voor network authentication met andere credentials
::

### Remote Desktop Sessions

**Remote Desktop (RDP)** creëert een **volledig interactive session** over het netwerk.

::note
**RDP Session Characteristics:**
- **Logon Type 10** in Security Event Log (Event ID 4624)
- Volledige GUI environment over het netwerk
- Network Level Authentication (NLA) vereist credentials vóór session establishment
- Kan restricted admin mode gebruiken voor credential protection
::

## 🤖 Non-Interactive Sessions

**Non-interactive accounts** verschillen van standaard user accounts omdat ze **geen login credentials vereisen**. Deze accounts worden voornamelijk gebruikt door het Windows besturingssysteem zelf.

### Non-Interactive Account Types

Windows heeft **drie primary non-interactive account types**:

| Account | Identity | Beschrijving |
|---------|----------|--------------|
| **Local System** | `NT AUTHORITY\SYSTEM` | Meest krachtige account in Windows — meer privileges dan local administrators |
| **Local Service** | `NT AUTHORITY\LocalService` | Minder geprivilegieerde versie van SYSTEM — vergelijkbaar met lokale user account |
| **Network Service** | `NT AUTHORITY\NetworkService` | Vergelijkbaar met domain user account — kan authenticated network sessions opzetten |

## 🔄 Session Logon Types

Windows onderscheidt verschillende **logon types** die in Event Viewer (Event ID 4624) worden gelogd:

| Logon Type | Naam              | Beschrijving                                             |
| ---------- | ----------------- | -------------------------------------------------------- |
| **2**      | Interactive       | Lokale keyboard/screen logon                             |
| **3**      | Network           | Netwerk verbinding (SMB shares, remote file access)      |
| **4**      | Batch             | Scheduled task execution                                 |
| **5**      | Service           | Service startup                                          |
| **7**      | Unlock            | Workstation unlock (screensaver)                         |
| **8**      | NetworkCleartext  | Network logon met cleartext credentials (IIS Basic Auth) |
| **9**      | NewCredentials    | RunAs /netonly — credentials voor network access         |
| **10**     | RemoteInteractive | Remote Desktop (RDP) logon                               |
| **11**     | CachedInteractive | Logon met cached domain credentials (offline)            |
