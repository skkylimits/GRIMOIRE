## 💻 Desktop Experience vs. Server Core

Windows Server Core werd voor het eerst uitgebracht met Windows Server 2008 als een minimalistische Server-omgeving die alleen de belangrijkste Server-functionaliteit bevat. Als resultaat heeft Server Core lagere beheerseisen, een kleiner aanvalsoppervlak en gebruikt het minder schijfruimte en geheugen dan zijn Desktop Experience (GUI) tegenhanger.

In Server Core worden alle configuratie- en onderhoudstaken uitgevoerd via de **command-line**, **PowerShell**, of **remote management** met MMC of Remote Server Administration Tools (RSAT).

::note
Server Core richt zich op een kleinere footprint door het ontbreken van een GUI, maar sommige grafische programma's worden nog steeds ondersteund, zoals Registry Editor, Notepad, System Information, Windows Installer, Task Manager en PowerShell.
::

### 🛠️ Ondersteunde Sysinternals Tools

Server Core ondersteunt ook sommige tools uit de Sysinternals suite:
- Active Directory Explorer
- Process Explorer
- Process Monitor
- TCPView

### ⚙️ Initiële Setup met Sconfig

Vanaf Windows Server 2019 moet **Server Core** of **Desktop Experience** geselecteerd worden tijdens de installatie, en geen van beide kan worden teruggedraaid (d.w.z. het converteren van Server Core naar Desktop Experience is niet mogelijk).

![sconfig](/content/7.knowledge-base/4.operating-systems/10.windows/1.fundamentals/12.desktop-vs-server/sconfig.png)

::important
Eenmaal geïnstalleerd kan de initiële setup voor Server Core worden uitgevoerd via **Sconfig**, een tekstgebaseerde interface (eigenlijk een VBScript uitgevoerd door WScript).
::

**Sconfig** wordt gebruikt voor het uitvoeren van verschillende veelvoorkomende taken:
- Netwerkconfiguratie
- Controleren op/installeren van Windows updates
- Accountbeheer
- Remote management configuratie
- Windows activeren
- En meer...

### ⚠️ Beperkingen

Bepaalde server-applicaties kunnen **niet** draaien op Server Core:
- Microsoft Server Virtual Machine Manager 2019 (SCVMM)
- System Center Data Protection Manager 2019
- SharePoint Server 2019
- Project Server 2019

::caution
Server Core is lichter en minder resource-intensief, maar heeft een steilere leercurve en kan moeilijker te beheren zijn. Het heeft ook enkele beperkingen, zoals het uitvoeren van beheertaken met bepaalde GUI-programma's.
::

## 🎯 Wanneer Gebruik Je Wat?

In elke omgeving moet de keuze tussen Desktop Experience of Server Core voor een server-installatie worden bepaald door:
- De **zakelijke behoefte**
- Het **beoogde gebruik** van de server
- Het **vaardigheidsniveau** van de beheerders die verantwoordelijk zijn voor het onderhoud
