**Disclaimer:**
The purpose is to guide the student through the process of performing digital forensic analysis of various Windows artifacts.
It does **not** represent an actual template of a forensic report.

## **System Information**

### **Computername:**
- **Registry Path:**
  `HKLM\System\CurrentControlSet\Control\Computername\`

### **Windows Version:**
- **Registry Path:**
  `HKLM\Software\Microsoft\Windows NT\Currentversion\`

### **Timezone:**
- **Registry Path:**
  `HKLM\System\CurrentControlSet\Control\TimeZoneInformation\`

### **Network Information:**
- **Registry Path:**
  `HKLM\System\CurrentControlSet\Services\Tcpip\Parameters\Interfaces\{interface-name}`

### **Shutdown Time:**
- **Registry Path:**
  `HKLM\System\ControlSet001\Control\Windows\ShutdownTime`

### **Defender Settings:**
- **Registry Path:**
  `HKLM\Software\Microsoft\Windows Defender\`

## **Users, Groups, and User Profiles**

### **Active Accounts during Attack Timeframe:**
- **Questions:**
  - Which account(s) were created?
  - Which accounts are Administrator group members?
  - Which users have profiles?

## **User Behavior**

- **UserAssist:** Applications opened
- **RecentDocs:** Files and folders opened
- **Shellbags:** Locations browsed by the user
- **Open / Save MRU:** Files that were opened
- **Last-Visited MRU:** Applications used to open files

## **NTFS - File System Analysis**

- **File Locations:**
  - Path: `My Computer\CLSID_Desktop\PWF-main\PWF-main\AtomicRedTeam`
- **MFT Analysis:**
  - Entry Number for `ART-attack.ps1`?
  - MACB Timestamps for `ART-attack.ps1`?
  - Was `ART-attack.ps1` timestomped?
- **File Deletion Details:**
  - Creation and deletion time for `deleteme_T1551.004`?
  - MFT Entry number for `deleteme_T1551.004` and current existence?

## **Execution Artifacts**

### **Background Activity Moderator (BAM)**
- **Registry Path:**
  `HKLM\SYSTEM\CurrentControlSet\Services\bam\UserSettings`
- **Recorded Executables:**
  - Which `.exe` files were recorded for **IEUser (RID 1000)**?
  - Include last execution date and time.

### **Application Compatibility Cache (AppCompatCache)**
- **Registry Path:**
  `SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache`
- **Cache Entry Positions:**
  - `AtomicService.exe:`
  - `mavinject.exe:`

### **AmCache**
- **Registry Path:**
  `C:\Windows\AppCompat\Programs\Amcache.hve`
- **SHA-1 Hash for AtomicService.exe?**

### **Prefetch Files**
- **Path:**
  `C:\Windows\Prefetch\*.pf`
- **Prefetch Timeline Analysis:**
  - Analyze suspicious execution events using **Eric Zimmerman's Timeline Explorer**.
  - Focus on:
    - POWERSHELL.exe
    - cmd.exe
    - NET.exe
    - REG.exe
    - SCHTASKS.exe
    - SC.exe
    - ATOMICSERVICE.EXE
    - MAVINJECT.exe
    - NOTEPAD.exe

## **Shortcut (LNK) Files**

- **Paths:**
  - `C:\users\<username>\AppData\Roaming\Microsoft\Windows\Recent`
  - `C:\users\<username>\AppData\Roaming\Microsoft\Office\Recent`

## **Persistence Mechanisms**

### **Auto-Run Keys**
- **Registry Paths:**
  - `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
  - `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\RunOnce`
  - `HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run`
  - `HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\RunOnce`
- **Analysis Focus:**
  - Full path of `AtomicService.exe` added to run keys?

### **Startup Folder**
- **Paths:**
  - `C:\Users\[Username]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
  - `C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp`
- **Analysis Focus:**
  - Name of suspicious script in the Startup folder?

## **Windows Services**

- **Registry Path:**
  `HKLM\SYSTEM\CurrentControlSet\Services`
- **Analysis Focus:**
  - Installation time of suspicious **AtomicService**?

## **Scheduled Tasks**

- **Registry Paths:**
  - `HKLM\Software\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tasks`
  - `HKLM\Software\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tree`
- **Path:**
  `C:\Windows\System32\Tasks`
- **Analysis Focus:**
  - Tasks created by **IEUser**?
  - Creation time?
  - Number of executions?
