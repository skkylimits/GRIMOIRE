---
title: Check-MailboxPermissions
Description: Dit PowerShell-script controleert de Full Access, Send As en SendOnBehalf rechten van een gebruiker op alle mailboxen in Exchange Online en exporteert de resultaten naar een CSV-bestand.
---

::caution
Send-on-behalf werkt nog niet naar behoren. Verder testen.
::

## 🤔 Waarom is het “traag”?

De reden dat het script langzaam is, is omdat:

1. **`Get-MailboxPermission`** werkt alleen per mailbox. Dit betekent dat je voor elke mailbox moet controleren of een specifieke gebruiker rechten heeft.

2. **Exchange Online** biedt geen directe cmdlet zoals:
   *"Geef mij alle mailboxen waar gebruiker X toegang op heeft."*

3. Helaas slaat Exchange de rechten op **per mailbox**, niet **per gebruiker**. Dit betekent dat je mailboxen moet doorlopen om te zien wie er toegang heeft — dit proces is traag omdat je per mailbox moet controleren, in plaats van dat je een overzicht krijgt van alle mailboxen waartoe een gebruiker toegang heeft.

## 👨‍💻 Script:

```powershell
$User = "simone.duin@meerwaarde.nl"
$Result = @()

Write-Host "Start met controleren van mailboxrechten voor: $User" -ForegroundColor Cyan

Get-Mailbox -ResultSize Unlimited | ForEach-Object {
    $mbx = $_
    $userHasFullAccess = $false
    $userHasSendAs = $false
    $userHasSendOnBehalf = $false

    Write-Host "`nControleren: $($mbx.DisplayName) <$($mbx.UserPrincipalName)>" -ForegroundColor Yellow

    # --- Full Access ---
    try {
        $fullAccessPerms = Get-MailboxPermission -Identity $mbx.UserPrincipalName | Where-Object {
            $_.User -like $User -and $_.AccessRights -contains "FullAccess" -and $_.IsInherited -eq $false
        }

        if ($fullAccessPerms) {
            Write-Host "✅ Full Access gevonden" -ForegroundColor Green
            $userHasFullAccess = $true
        } else {
            Write-Host "⛔ Geen Full Access" -ForegroundColor DarkGray
        }
    }
    catch {
        Write-Host "❌ Fout bij ophalen Full Access permissies: $_" -ForegroundColor Red
    }

    # --- Send-As ---
    try {
        $sendAsPerms = Get-RecipientPermission -Identity $mbx.UserPrincipalName -ErrorAction Stop | Where-Object {
            $_.Trustee -like $User -and $_.AccessRights -contains "SendAs"
        }

        if ($sendAsPerms) {
            Write-Host "📨 Send-As gevonden" -ForegroundColor Cyan
            $userHasSendAs = $true
        } else {
            Write-Host "⛔ Geen Send-As" -ForegroundColor DarkGray
        }
    }
    catch {
        Write-Host "❌ Fout bij ophalen Send-As permissies: $_" -ForegroundColor Red
    }

    # --- SendOnBehalf ---
    try {
        $sendOnBehalfPerms = Get-MailboxPermission -Identity $mbx.UserPrincipalName | Where-Object {
            $_.User -like $User -and $_.AccessRights -contains "SendOnBehalf" -and $_.IsInherited -eq $false
        }

        if ($sendOnBehalfPerms) {
            Write-Host "📤 SendOnBehalf gevonden" -ForegroundColor Magenta
            $userHasSendOnBehalf = $true
        } else {
            Write-Host "⛔ Geen SendOnBehalf" -ForegroundColor DarkGray
        }
    }
    catch {
        Write-Host "❌ Fout bij ophalen SendOnBehalf permissies: $_" -ForegroundColor Red
    }

    # --- Toegang status bepalen ---
    if ($userHasFullAccess -and $userHasSendAs -and $userHasSendOnBehalf) {
        $accessType = "FullAccess + SendAs + SendOnBehalf"
    } elseif ($userHasFullAccess -and $userHasSendAs) {
        $accessType = "FullAccess + SendAs"
    } elseif ($userHasFullAccess -and $userHasSendOnBehalf) {
        $accessType = "FullAccess + SendOnBehalf"
    } elseif ($userHasSendAs -and $userHasSendOnBehalf) {
        $accessType = "SendAs + SendOnBehalf"
    } elseif ($userHasFullAccess) {
        $accessType = "Alleen FullAccess"
    } elseif ($userHasSendAs) {
        $accessType = "Alleen SendAs"
    } elseif ($userHasSendOnBehalf) {
        $accessType = "Alleen SendOnBehalf"
    } else {
        $accessType = "Geen toegang"
    }

    # --- Resultaat toevoegen aan array ---
    $Result += [PSCustomObject]@{
        Mailbox        = $mbx.DisplayName
        MailboxType    = $mbx.RecipientTypeDetails
        UserChecked    = $User
        FullAccess     = if ($userHasFullAccess) { "✅" } else { "❌" }
        SendAs         = if ($userHasSendAs) { "✅" } else { "❌" }
        SendOnBehalf   = if ($userHasSendOnBehalf) { "✅" } else { "❌" }
        ToegangStatus  = $accessType
    }
}

# --- Export naar CSV ---
$csvPath = "C:\Temp\MailboxRechten_$($User.Split('@')[0])_$(Get-Date -Format 'yyyyMMdd_HHmm').csv"
$Result | Export-Csv -Path $csvPath -NoTypeInformation -Encoding UTF8

Write-Host "`n✅ CSV geëxporteerd naar: $csvPath" -ForegroundColor Green
Write-Host "Klaar met controleren." -ForegroundColor Cyan
```

## 📂 CSV Export Locatie:
Het bestand wordt geëxporteerd naar **`C:\Temp`** en bevat de volgende kolommen:

| Mailbox       | MailboxType    | UserChecked              | FullAccess | SendAs | SendOnBehalf | ToegangStatus          |
|---------------|----------------|--------------------------|------------|--------|--------------|------------------------|
| Klant Service | SharedMailbox  | simone.duin@meerwaarde.nl | ✅         | ❌     | ✅           | FullAccess + SendOnBehalf |
| Info          | UserMailbox    | simone.duin@meerwaarde.nl | ✅         | ✅     | ❌           | FullAccess + SendAs    |
| HRM           | SharedMailbox  | simone.duin@meerwaarde.nl | ❌         | ❌     | ❌           | Geen toegang          |
