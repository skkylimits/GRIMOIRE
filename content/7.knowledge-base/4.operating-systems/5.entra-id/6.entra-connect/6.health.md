**Microsoft Entra Connect Health** offers centralized monitoring for your on-prem identity infrastructure. It ensures reliable connectivity to Microsoft 365 by tracking key identity components. The portal provides alerts, performance stats, and usage analytics—all in one place.

![aadconnecthealth2](/content/7.knowledge-base/4.operating-systems/5.entra-id/6.entra-connect/6.health/aadconnecthealth2.png)

## Why

**Microsoft Entra Connect Health** helps keep your sign-ins working by checking the health of your on-premises servers. It makes sure users can access both cloud and local apps without issues. Just install a small tool on your servers (like AD FS), and it gives you useful info and alerts to spot problems early.

| **Key Benefits**          | **Best Practices**                                              |
| ------------------------- | --------------------------------------------------------------- |
| **Enhanced Security**     | Extranet lockout trends, failed sign-ins, alerts on ADFS issues |
| **Easy to Manage**        | Quick agent install, auto-updates, fast data visibility         |
| **Rich Insights**         | App usage, network locations, server token requests             |
| **Great User Experience** | Central dashboard, email alerts for key events                  |
