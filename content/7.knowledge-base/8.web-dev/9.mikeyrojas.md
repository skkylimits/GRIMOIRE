## 🧩 Code Flow

### `index.js`

```js
import Experience from './Experience/Experience'
import './style.css'

const experience = new Experience(document.querySelector('canvas.webgl'))
```

* **Loads global styles**
* **Initializes** the main `Experience` class with the `<canvas>` element

### `Experience.js`

Responsible for:

* Initializing scene, renderer, camera
* Managing update/render loop
* Delegating tasks to modules like `Camera`, `World`, etc.

```js
constructor(canvas) {
  this.canvas = canvas
  this.scene = new THREE.Scene()
  this.camera = new Camera(this)
  this.renderer = new Renderer(this)
  this.world = new World(this)
  ...
}
```
