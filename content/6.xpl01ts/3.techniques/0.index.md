This is an attempt to map out all different techniques used by attackers to gain unauthorized access to systems. The techniques are categorized based on the tools and methods used to exploit vulnerabilities and bypass security mechanisms.

1. **LOLBins (Living Off The Land Binaries)**
   - PowerShell, Cmd.exe, WMI, MSHTA, Regsvr32.

2. **Living Off The Land (LOTL)**
   - Scheduled Tasks, Windows Services, DLL Injection.

3. **Abuse Elevation Control Mechanisms**
   - UAC bypass, Token Impersonation.

4. **Sudo and Sudo Caching (Linux/MacOS)**
   - Exploiting Sudo permissions for elevated privileges.

5. **WMI (Windows Management Instrumentation)**
   - Remote execution, lateral movement.

6. **Powershell Empire**
   - Post-exploitation framework using PowerShell.

7. **MSBuild**
   - Abusing MSBuild for malicious code execution.

8. **Script-based Techniques**
   - VBScripts, Batch Scripts, PowerShell Scripts.

9. **Phishing via Legitimate Applications**
   - Office Macros, Macro-enabled documents.

10. **Abuse of Cloud Services**
   - AWS Lambda, Azure Automation, Google Cloud Functions.

11. **Registry-based Techniques**
   - Exploiting the Windows Registry for persistence.

12. **Abuse of System Services**
   - ServiceDLL, Service Account Exploits.
