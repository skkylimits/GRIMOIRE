## **What is Azure?**

- **Azure** is **Microsoft’s cloud computing platform**, similar to AWS (Amazon) and GCP (Google).
- Offers **200+ cloud products & services**, with continuous expansion.
- **Trusted by 95% of Fortune 500** companies for cloud solutions.

## **Azure Services**

- **Compute** – VMs, Kubernetes, Containers, Function Apps
- **Networking** – VNet, VPN Gateway, Load Balancing, CDN
- **Storage** – Blob, File, Queue, Table
- **Mobile** – Backend services
- **Databases** – Cosmos DB, Managed SQL, MySQL, PostgreSQL
- **Web** – App Service, API Management, Cognitive Search
- **IoT** – Internet of Things solutions
- **Big Data** – Scalable analytics & processing
- **AI** – Machine learning & cognitive services
- **DevOps** – CI/CD, monitoring, automation
