## **Microsoft Tools**
- **Az CLI**: [Command-line interface](https://learn.microsoft.com/en-us/cli/azure/) for managing Azure resources.
- **Az PowerShell Module**: [The updated PowerShell module](https://learn.microsoft.com/en-us/powershell/azure/new-azureps-module-az?view=azps-13.1.0) for managing Azure resources, replacing the older AzureRM and Azure modules.
- **AzureAD PowerShell Module**: Used for managing Entra ID ([this](https://techcommunity.microsoft.com/blog/microsoft-entra-blog/important-update-deprecation-of-azure-ad-powershell-and-msonline-powershell-modu/4094536) will be supported until June 30th, 2025).
- **Microsoft Graph Module (Mg module)**: For interacting with [Microsoft Graph APIs](https://developer.microsoft.com/en-us/graph/graph-explorer).
- **Microsoft Portals**: A range of online portals to manage Azure and Entra ID resources. A comprehensive list can be found at [msportals.io](https://msportals.io/).
