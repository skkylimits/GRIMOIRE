title: How To Hack Firewalls
description: Step by step guideline to control the firewall
---

1. Physical access? Y or N.
If yes, then gain access and plant device of choice and test for outside access.

2. If remote, scan the shit out of it using a mullvad owned vpn, try to find any brand, model, and version info. 

3. Scan ports. All of them. Using different kinds of scans, multiple scanners like Nmap and others for different results. Scan udp overnight, it takes forever.

4. Compile findings, and use vulnerability scans based on findings abd what ports are open. 

5. If login portal is open, mark as possible brute force target

6.try to search for brand specific misconfigurations as well. 

6. If all else fails, do more research on target, see if you can find out if they use a company portal, or vpn access of some kind. Enumerate and rework plan based on findings.
