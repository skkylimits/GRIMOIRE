## Ubuntu in VM

0. Get ubuntu up and running in VM - https://ubuntu.com/tutorials/how-to-run-ubuntu-desktop-on-a-virtual-machine-using-virtualbox#1-overview

```
# Install VirtualBox Guest Additions

$ sudo apt-get update
$ sudo apt-get upgrade

# Then install the necessary packages to build the kernel modules and approve the additional disk space needed.

$ sudo apt install build-essential dkms linux-headers-$(uname -r)

# Restart the VM.

$ reboot
```

```
# If the virtual disk won't run. Try manually running the installation.
# https://virtual-dba.com/blog/issues-installing-guest-additions-ubuntu-20-04/

$ cd /media/username/VBOX_GAs_6.1.38

$ sudo ./VBoxLinuxAdditions.run`

```

## MOBSF

1. Install MOBSF https://mobsf.github.io/docs/#/installation

Run it with docker on http://127.0.0.1:8000/

```
$ docker pull opensecurity/mobile-security-framework-mobsf

# Static Analysis Only

$ docker run -it --rm -p 8000:8000 opensecurity/mobile-security-framework-mobsf:latest
```
