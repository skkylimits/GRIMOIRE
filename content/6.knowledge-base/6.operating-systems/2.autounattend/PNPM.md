Title: PNPM
---

### 1. Zorg dat Node.js en npm geïnstalleerd zijn

(Als je nvm gebruikt, zorg dan dat je een Node-versie actief hebt, bijvoorbeeld: `nvm use --lts`)

### 2. Installeer pnpm via npm

```bash
npm install -g pnpm
```

### 3. Controleer of pnpm goed is geïnstalleerd

```bash
pnpm --version
```
