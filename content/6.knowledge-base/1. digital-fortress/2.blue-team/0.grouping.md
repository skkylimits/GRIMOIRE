### 1. **Strategisch Beheer en Governance**

- **Governance, Risk Management, and Compliance (GRC)**
  - NIST Cybersecurity Framework
  - ISO/IEC 27001
  - COBIT
  - ITIL
  - ISF Standard of Good Practice
  - GDPR
  - HIPAA
  - PCI DSS
  - BSI Grundschutz
  - CMMC
  - ITSG-33
  - ISO/IEC 15408
  - NICE Cybersecurity Workforce Framework

### 2. **Beveiligingstests en Evaluaties**

- **Security Testing and Assessment Methodologies**
  - ISSAF (Information Systems Security Assessment Framework)
  - OSSTMM (Open Source Security Testing Methodology Manual)
  - PTES (Penetration Testing Execution Standard)
  - CIS Controls
  - SANS Critical Security Controls
  - CWE (Common Weakness Enumeration)
  - CAPEC (Common Attack Pattern Enumeration and Classification)
- **Red Teaming**
  - Penetration Testing
  - Vulnerability Assessment
  - Attack Simulation

### 3. **Incidentbeheer en Dreigingsinformatie**

- **Threat Intelligence and Incident Management**
  - MITRE ATT&CK Framework
  - Cyber Kill Chain
  - VERIS (Vocabulary for Event Recording and Incident Sharing)

### 4. **Operationele Beveiliging (Blue Teaming)**

- **Defensive Security (Blue Teaming)**
  - Network Segmentation
  - System Hardening
  - Secure Software Development Lifecycle (SDLC)
  - Encryption Key Management
  - Identity Governance and Administration (IGA)
  - Privileged Access Management (PAM)
  - Zero Trust Network Access (ZTNA)
  - Data Loss Prevention (DLP)
  - Security Automation and Orchestration
  - Behavioral Analytics
- **Identity and Access Management**
  - Multi-Factor Authentication (MFA)
  - Single Sign-On (SSO)
  - Role-Based Access Control (RBAC)
  - Attribute-Based Access Control (ABAC)
  - Federated Identity Management
  - Access Control Mechanisms
- **Beveiliging en Risicobeheer**
  - Risk Assessment and Management Frameworks
    - FAIR (Factor Analysis of Information Risk)
    - CRAMM (CCTA Risk Analysis and Management Method)
    - FAIR (Framework for Assessing and Improving Cybersecurity Resilience)
    - MITRE Shield
- **Toepassings- en Infrastructuurbeveiliging**
  - **Web Application Security Frameworks**
    - OWASP Top 10
    - OWASP API
  - **Mobiele en Cloudbeveiliging**
    - OWASP Mobile
    - OWASP Cloud
    - OWASP IoT
  - **Cloud and IoT Security Frameworks**
    - CSA Security Guidance

### 5. **Werkrolontwikkeling en -beheer**

- **Workforce and Role Frameworks**
  - NICE Cybersecurity Workforce Framework
