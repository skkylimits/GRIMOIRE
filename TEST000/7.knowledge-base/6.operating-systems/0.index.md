## **Voor Systeembeheerders**

### **Systeemconfiguratie en Beleid**
1. **gpedit.msc** - Lokale Groepsbeleidseditor
2. **msconfig** - Systeemconfiguratie
3. **secpol.msc** - Lokale Beveiligingsbeleid

### **Beheer van Systeemcomponenten**
4. **compmgmt.msc** - Computerbeheer
5. **diskmgmt.msc** - Schijfbeheer
6. **services.msc** - Services

### **Netwerk en Gebruikersbeheer**
7. **dsa.msc** - Active Directory-gebruikers en -computers
8. **certmgr.msc** - Certificaatbeheer
9. **control userpasswords2** - Gebruikersaccounts

### **Diagnostiek en Prestatiemonitoring**
10. **eventvwr** - Event Viewer (Gebeurtenislogboeken)
11. **perfmon** - Prestatiemonitor
12. **dxdiag** - DirectX Diagnostic Tool
13. **msinfo32** - Systeeminfo

### **Overige Beheerhulpmiddelen**
14. **regedit** - Register-editor
15. **taskmgr** - Taakbeheer
16. **mmc** - Microsoft Management Console

## **Voor Algemeen Systeemgebruik**

1. **temp** - Tijdelijke bestanden map (`%temp%`)
2. **wsreset.exe** - Reset Windows Store Cache
3. **control** - Configuratiescherm
4. **cmd** - Opdrachtprompt
5. **calc** - Rekenmachine
6. **notepad** - Kladblok
7. **winver** - Windows Versie weergeven
8. **shutdown /s /t 0** - Computer uitschakelen
9. **shutdown /r /t 0** - Computer herstarten
10. **shutdown /l** - Gebruiker afmelden

## Windows Commands

PowerShell Cmdlets with corresponding CMD commands for quick reference.

### **PowerShell Cmdlets for CMD Commands:**

| **CMD Command**                | **PowerShell Cmdlet**                                                  | **What it Does**                                                                 |
|---------------------------------|--------------------------------------------------------------------------|---------------------------------------------------------------------------------|
| `ipconfig`                      | `Get-NetIPAddress`                                                       | Displays the IP configuration of all network interfaces (IP address, subnet mask, gateway). |
| `ipconfig /all`                 | `Get-NetIPConfiguration`                                                 | Provides detailed information about network adapters, including DNS and Gateway. |
| `findstr`                       | `Select-String`                                                         | Searches for a specified string in files or output, similar to `findstr`.        |
| `ipconfig /release`             | `Invoke-WmiMethod -Class Win32_NetworkAdapterConfiguration -Name ReleaseDHCPLeaseAll` | Releases the DHCP lease for all network adapters, disconnecting from the network. |
| `ipconfig /renew`               | `Invoke-WmiMethod -Class Win32_NetworkAdapterConfiguration -Name RenewDHCPLeaseAll`   | Renews the DHCP lease for all network adapters, re-establishing the network connection. |
| `ipconfig /displaydns`          | `Get-DnsClientCache`                                                     | Displays the DNS cache with recently resolved domain names. Use `| Format-List` to show all columns. |
| `clip`                          | `Set-Clipboard`                                                         | Copies text or data to the clipboard, useful for pasting elsewhere.              |
| `ipconfig /flushdns`            | `Clear-DnsClientCache`                                                   | Clears the DNS client cache, removing all cached DNS entries.                    |
| `nslookup`                      | `Resolve-DnsName`                                                       | Resolves a hostname into an IP address (forward lookup) or vice versa (reverse lookup). |
| `cls`                           | `Clear-Host` (or **Ctrl+L**)                                              | Clears the terminal screen for a clean slate.                                   |
| `getmac /v`                     | `Get-NetAdapter`                                                        | Displays detailed information about network adapters, such as MAC address and status. |
| `powercfg`                      | No direct equivalent in PowerShell                                        | Used to configure power settings on Windows. Use `Get-Command` to explore power management cmdlets. |
| `assoc`                         | No direct equivalent in PowerShell                                        | Displays or changes file type associations. PowerShell can manage file associations via `New-ItemProperty` and `Set-ItemProperty`. |
| `chkdsk`                        | `Repair-Volume`                                                          | Scans and repairs disk volumes for errors, similar to `chkdsk`.                  |
| `sfc`                           | No direct equivalent in PowerShell                                        | Scans system files for integrity violations and repairs corrupted files.         |
| `DISM`                          | No direct equivalent in PowerShell                                        | Used to service and repair Windows images.                                       |
| `tasklist`                      | `Get-Process`                                                           | Lists all currently running processes on the system.                             |
| `taskkill`                      | `Stop-Process`                                                          | Terminates a running process by name or PID.                                     |
| `netsh`                         | No direct equivalent in PowerShell                                        | Configures network settings like IP addresses and firewalls. Use cmdlets like `Get-NetFirewallRule` for firewall management. |
| `ping`                          | `Test-NetConnection`                                                     | Sends ICMP echo requests to test connectivity and measure latency.               |
| `ping /t`                       | `Test-Connection -Count 100000`                                          | Performs continuous ping tests to a target host, showing latency and packet loss. |
| `tracert`                       | `Test-NetConnection -TraceRoute`                                         | Traces the route packets take to reach a destination, showing intermediate hops. |
| `netstat`                       | `Get-NetTCPConnection`                                                   | Displays active network connections and open ports.                              |
| `route print`                   | `Get-NetRoute`                                                          | Displays the current IP routing table, showing how packets are forwarded.        |
| `route add`                     | `New-NetRoute`                                                          | Adds a new route to the routing table, specifying how packets should be forwarded. |
| `route delete`                  | `Remove-NetRoute`                                                       | Removes a specific route from the IP routing table.                              |
| `shutdown`                      | `Stop-Computer`                                                         | Shuts down the computer, either immediately or with a delay.                     |
| `restart`                       | `Restart-Computer`                                                      | Restarts the computer.                                                          |
