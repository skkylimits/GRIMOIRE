## Wat zijn Structures (Structs)?

Stel je voor dat je een doos hebt waar je verschillende dingen in wilt stoppen: een potlood, een gom, een liniaal. Ze horen bij elkaar.

In C kun je zo’n "doos" maken met een **struct**. Een **struct** groepeert data van verschillende soorten in één pakketje. Elk stukje in de doos heet een **member** of **element**.

Een voorbeeld van zo’n struct die Windows gebruikt:

```c
typedef struct tagTHREADENTRY32 {
  DWORD dwSize;            // Member 1
  DWORD cntUsage;          // Member 2
  DWORD th32ThreadID;
  DWORD th32OwnerProcessID;
  LONG  tpBasePri;
  LONG  tpDeltaPri;
  DWORD dwFlags;
} THREADENTRY32; 
```

👉 Je hoeft nu nog niet te snappen wat al die members doen, belangrijk is dat je ziet: **het is gewoon een bundeltje variabelen**.

## Hoe declareer je een Structure?

Meestal gebruiken we het **typedef** keyword om een alias te geven.

```c
typedef struct _STRUCTURE_NAME {
  // structure elements
} STRUCTURE_NAME, *PSTRUCTURE_NAME;
```

* `STRUCTURE_NAME` = de struct zelf.
* `PSTRUCTURE_NAME` = een pointer naar de struct (de `P` staat voor *Pointer*).

## Initialiseren van een Struct

Je kunt een struct direct vullen, of eerst een pointer maken en dan vullen.

```c
STRUCTURE_NAME struct1 = { 0 };  
// Alles in struct1 staat nu op nul
```

Of met de pointer:

```c
PSTRUCTURE_NAME structpointer = NULL;
```

## Leden (members) gebruiken

Met de **dot operator (.)** vul je members:

```c
typedef struct _STRUCTURE_NAME {
  int ID;
  int Age;
} STRUCTURE_NAME, *PSTRUCTURE_NAME;

STRUCTURE_NAME struct1 = { 0 };
struct1.ID   = 1470;
struct1.Age  = 34;
```

Met een **pointer** gebruik je de **arrow operator (->)**:

```c
PSTRUCTURE_NAME structpointer = &struct1;
structpointer->ID = 8765;
printf("ID is nu: %d \n", structpointer->ID);
```

👉 `structpointer->ID` is hetzelfde als `(*structpointer).ID`.

## Enums (Enumeration)

Een **enum** is gewoon een lijstje met namen die elk een nummer krijgen. Bijvoorbeeld:

```c
enum Weekdays {
  Monday,    // 0
  Tuesday,   // 1
  Wednesday, // 2
  Thursday,  // 3
  Friday,    // 4
  Saturday,  // 5
  Sunday     // 6
};
```

Als je dan zegt:

```c
enum Weekdays Today = Friday;
```

Dan is `Today` eigenlijk gewoon 4.
Met een `switch` kun je die checken:

```c
switch (Today) {
  case Monday:    printf("It's Monday!\n"); break;
  case Friday:    printf("It's Friday!\n"); break;
}
```

## Union

Een **union** lijkt op een struct, maar alle members delen **dezelfde memory plek**. Dus je kunt maar één tegelijk gebruiken.

```c
union ExampleUnion {
   int    IntegerVar;
   char   CharVar;
   float  FloatVar;
};
```

👉 Als je `IntegerVar` vult, dan verandert automatisch `CharVar` en `FloatVar` ook (omdat ze in dezelfde memory zitten).

## Bitwise Operators

Bitwise operators werken niet met hele getallen, maar met de **bitjes** zelf.

* **Right shift (>>)** – schuift bits naar rechts.
* **Left shift (<<)** – schuift bits naar links.
* **OR (|)** – zet een bit op 1 als één van beide bits 1 is.
* **AND (&)** – zet een bit alleen op 1 als beide 1 zijn.
* **XOR (^)** – zet een bit op 1 als ze verschillend zijn.
* **NOT (\~)** – keert alle bits om.

Voorbeeld Right Shift:

```
10100111 >> 2  →  00101001
```

Voorbeeld Left Shift:

```
10100111 << 2  →  10011100
```

## Passing by Value

Als je een waarde meegeeft **by value**, maak je eigenlijk een kopie. De originele blijft onaangetast.

```c
int add(int a, int b) {
   return a + b;
}

int main() {
   int x = 5;
   int y = 10;
   int sum = add(x, y); // x en y blijven hetzelfde
}
```

## Passing by Reference

Als je iets meegeeft **by reference**, geef je eigenlijk het **adres (pointer)** mee. Dan kan de functie de originele waarde aanpassen.

```c
void add(int *a, int *b, int *result) {
   *result = *a + *b;
}

int main() {
   int x = 5;
   int y = 10;
   int sum = 0;

   add(&x, &y, &sum);  
   // sum = 15
}
```


👉 **Samenvatting:**

* Structs = dozen met verschillende variabelen.
* Enums = lijstjes met namen.
* Unions = verschillende variabelen delen dezelfde memory.
* Bitwise = speel met bits.
* Passing by value = kopie → veilig.
* Passing by reference = pointer → origineel verandert.

Bitwise OR

The bitwise OR operation is a logical operation that involves two binary values at the bit level. It evaluates each bit of the first operand against the corresponding bit of the second operand, generating a new binary value. The new binary value contains a 1 in any bit position where either one or both of the corresponding bits in the original values are 1.

Passing By Value

Passing by value is a method of passing arguments to a function where the argument is a copy of the object's value. This means that when an argument is passed by value, the value of the object is copied and the function can only modify its local copy of the object's value, not the original object itself.

int add(int a, int b)
{
   int result = a + b;
   return result;
}

int main()
{
   int x = 5;
   int y = 10;
   int sum = add(x, y); // x and y are passed by value

   return 0;
}


Passing By Reference

Passing by reference is a method of passing arguments to a function where the argument is a pointer to the object, rather than a copy of the object's value. This means that when an argument is passed by reference, the memory address of the object is passed instead of the value of the object. The function can then access and modify the object directly, without creating a local copy of the object.

void add(int *a, int *b, int *result)
{
  
  int A = *a; // A is now the same value of a passed in from the main function
  int B = *b; // B is now the same value of b passed in from the main function
  
  *result = B + A;
}

int main()
{
   int x = 5;
   int y = 10;
   int sum = 0;

   add(&x, &y, &sum);
  
   // 'sum' now is 15
   
   return 0;
}
