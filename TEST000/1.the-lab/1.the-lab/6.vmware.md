## Install a Hypervisor (VMware)

Download and install VMware to host your Windows/Kali Linux VM.

- **Installeer VMware Player**:
  - Download VMware Player van de officiële [VMware Player Download](https://www.vmware.com/go/getplayer-win)
  - Volg de installatie-instructies.

- **Installeer VMware Workstation Pro***:
   - Download VMware Workstation Pro van de officiële [VMware-website](https://www.vmware.com/products/workstation-player.html).
   - Volg de installatie-instructies.

## Install Kali

::steps{level="4"}

#### 1. Download Kali Linux VMware 64-bit
1. Ga naar de [Kali Linux Download-pagina](https://www.kali.org/get-kali/#kali-virtual-machines).
2. Klik op **Download** bij de **VMware 64-bit** versie.
3. Wacht tot de download is voltooid.

#### 2. Importeer de Kali VM in VMware
1. Open **VMware Workstation/Player**.
2. Klik op **Open a Virtual Machine**.
3. Zoek en selecteer het gedownloade **.ova-bestand**.
4. Klik op **Open** en vervolgens op **Import**.
5. Wacht tot de import is voltooid.

#### 3. Log in op Kali Linux
1. Start de Kali Linux VM in VMware.
2. Wacht tot het besturingssysteem is geladen.
3. Voer in bij **Username**: `kali`.
4. Voer in bij **Password**: `kali`.
5. Druk op **Enter** om in te loggen.

::

For additional help with installation, please read the [Kali.org docs](https://www.kali.org/docs/installation/).

## Install Windows

::steps{level="4"}

#### Download het Windows 11 ISO-bestand
   - Ga naar de [Microsoft Windows 11 downloadpagina](https://www.microsoft.com/en-us/software-download/windows11).
   - Kies "Download Windows 11 Disk Image (ISO)" en sla het bestand op je computer op.

#### VMware Player Starten
1. Open **VMware Player**.
2. Klik op **"Create a New Virtual Machine"**.

#### Kies de Configuratie
1. Selecteer **Custom (Advanced)** voor meer controle.
2. Klik op **Next**.

#### Virtual Machine Hardware Compatibility
1. Kies **VMware Workstation 16.x of hoger**.
2. Klik op **Next**.

#### Kies de Installatiemedia
1. Selecteer **Installer Disc Image File (ISO)**.
2. Zoek en selecteer het gedownloade **Windows 11 ISO-bestand**.
3. Controleer of **Windows 11 x64** wordt herkend.
4. Klik op **Next**.

#### Naam en Locatie
1. Geef je VM een naam of laat deze standaard.
2. Kies een opslaglocatie.
3. Klik op **Next**.

#### Encryption Type
1. Selecteer **Encrypt all the files**.
2. Voer een **8-cijferige pincode** in.
3. **Uncheck** de optie **Use Credential Manager**.
4. Klik op **Next**.

#### UEFI en Secure Boot
1. Selecteer **UEFI Firmware**.
2. Vink **Enable Secure Boot** aan.
3. Klik op **Next**.

#### CPU en Geheugen Instellen
1. **Processors**: Minimaal **2**, aanbevolen **3**.
2. **Cores** per processor: **3**.
3. **RAM**: **16,238 MB** aanbevolen.
4. Klik op **Next**.

#### Netwerk en Controller
1. Kies **Use NAT**.
2. Klik op **Next**.

#### I/O Controller Type
1. Selecteer **LSI Logic SAS**.
2. Klik op **Next**.

#### Disk Type
1. Kies **NVMe** voor beste prestaties.
2. Klik op **Next**.

#### Schijfinstellingen
1. Kies **Create a New Virtual Disk**.
2. Grootte: **Minimaal 64 GB**, aanbevolen **128 GB+**.
3. Selecteer **Store virtual disk as a single file**.
4. Klik op **Next**.

#### Disk Bestandsnaam
1. Laat de standaardnaam staan.
2. Klik op **Next**.

#### Hardwarecontrole
1. Controleer of:
   - **Windows 11 ISO** is geselecteerd.
   - **NAT-netwerkadapter** is ingeschakeld.
   - **UEFI en Secure Boot** aanstaan.
2. Klik op **Finish**.

####  VM Starten
1. Selecteer je nieuwe VM.
2. Klik op **Power on this virtual machine**.
3. Volg de Windows 11 installatieprocedure.

#### Problemen Oplossen
1. **Firmware aanpassen**: Start de VM en druk **ESC** voor het bootmenu.
2. **.VMX-bestand bewerken**:
   - Open het `.vmx`-bestand en controleer:

     ```plaintext
     ethernet0.present = "true"
     ethernet0.connectionType = "nat"
     ```

####  Windows Installatie Beginnen
1. **Selecteer taal, tijd en toetsenbordindeling** en klik op **Next**.
2. Klik op **Install Now**.
3. **Productcode**: Voer een code in of klik op **I don't have a product key**.
4. **Windows Versie**: Kies en klik op **Next**.
5. **Licentievoorwaarden**: Accepteer en klik op **Next**.
6. **Installatietype**: Kies **Custom: Install Windows only (advanced)**.
7. **Schijfselectie**: Selecteer de aangemaakte schijf en klik op **Next**.

####  Windows 11 Configuratie Voltooien
1. **Regio selecteren** > **Yes**.
2. **Toetsenbordindeling bevestigen** > **Yes**.
3. **Extra toetsenbordindeling?** > **Skip**.
4. **Apparaatnaam instellen (optioneel)** > **Next**.
5. **Inlogmethode kiezen**:
   - **Persoonlijk gebruik**: **Set up for personal use** > **Next**.
   - **Werk/school**: **Set up for work or school**.
6. **Offline Account selecteren** via **Sign-in options**.
7. **Gebruikersnaam invoeren** en klik op **Next**.
8. **Wachtwoord instellen en bevestigen**.
9. **Beveiligingsvragen invullen**.
10. **Privacyinstellingen configureren** > **Accept**.
11. **Windows configuratie voltooien** (dit kan enkele minuten duren).
12. **Bureaublad weergeven**: Je VM is klaar!
::

## VMware Tools
::steps{level="4"}

#### Open de **VMware Player**.
#### Ga naar **VM > Install VMware Tools**.
#### Open **Verkenner** in de VM.
#### Dubbelklik op de schijf **VMware Tools**.
#### Start **setup64.exe** (of **setup.exe** voor 32-bit).
#### Kies **Complete** en klik op **Install**.
#### Wacht tot de installatie voltooid is.
#### Klik op **Finish**.
#### Herstart de VM.

::

#### VMware Tools Voordelen
- **Volledig schermmodus**.
- **Gedeeld klembord** (kopiëren/plakken tussen host en VM).
- **Drag & Drop** bestanden tussen host en VM.
- **Snellere prestaties** en betere muisintegratie.

::tip
Resize de VM en kies **Keep Aspect Ratio Stretch** > **Free Stretch**.
::

##  Performance Optimalisatie
### NVMe M.2 vs HDD
- **NVMe**: **Sneller (3,000–7,000 MB/s), betere random access en IOPS**.
	- **NVMe** voor meerdere VMs en snelle prestaties.
- **HDD**: **Langzamer (50–150 MB/s), mechanische beperkingen**.
	- **HDD** als budget een rol speelt.

### VM Opslag: C:\ vs Partitie
- **Tijdelijk op C:\VM** mogelijk.
- **Beter om naar D:\VMs te verplaatsen**:
  1. Kopieer VM-bestanden naar nieuwe partitie.
  2. Update VMware-configuratie.
  3. Test de VM en verwijder oude bestanden.

### NVMe Fragmentatie & TRIM
- **Geen impact op prestaties** door:
  - **Geen mechanische delen**.
  - **TRIM-command optimaliseert blokbeheer**.
  - **Wear-leveling voorkomt slijtage**.
- **Optimalisatie Tips**:
  - Controleer of **TRIM is ingeschakeld** (`fsutil behavior query DisableDeleteNotify` moet `0` tonen).
  - Gebruik **Optimize Drives** in Windows voor TRIM.
  - Houd **10-20% vrije ruimte** aan.

::caution
**Geen defragmentatie** uitvoeren op NVMe.
::

### Conclusie
- **NVMe M.2** is het beste voor prestaties.
- **Verplaats VM’s naar een aparte partitie** voor betere organisatie.
- **TRIM en vrije ruimte** zijn belangrijker dan fragmentatie op NVMe.
