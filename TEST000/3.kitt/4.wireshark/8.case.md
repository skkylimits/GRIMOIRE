## ✅ Hostname wijzigen

1. **Bekijk huidige hostname**

   ```bash
   hostname
   ```

2. **Wijzig hostname**

   ```bash
   sudo hostnamectl set-hostname new-hostname
   ```

3. **Pas `/etc/hosts` aan indien nodig**

   ```bash
   sudo nano /etc/hosts
   ```

4. **Reboot**

   ```bash
   sudo reboot
   ```

## 🔑 GPG Key fout bij `apt-get update`

Tijdens `apt-get update` trad een GPG key error op:

```
W: GPG error: http://mirror.serverion.com/kali kali-rolling InRelease: The following signatures couldn't be verified because the public key is not available: NO_PUBKEY ED65462EC8D5E4C5
```

**Oplossing**:

```bash
curl -fsSL https://archive.kali.org/archive-key.asc | sudo gpg --dearmor -o /usr/share/keyrings/kali-archive-keyring.gpg
```

Voeg toe aan `sources.list`:

```
sudo nano /etc/apt/sources.list
```

```
deb [signed-by=/usr/share/keyrings/kali-archive-keyring.gpg] http://http.kali.org/kali kali-rolling main contrib non-free
```

Update opnieuw:

```bash
sudo apt-get update
```

**RTT-interpretatie:**

* < 30 ms → uitstekend
* 30–60 ms → goed
* 60–150 ms → acceptabel
* > 150 ms → merkbare vertraging
* > 300 ms → slecht, nauwelijks bruikbaar

## 🕵️ Verkeer vastleggen met `tcpdump`

1. **Bekijk netwerkinterfaces**:

   ```bash
   ip a
   ```

2. **Start capture op gewenste interface (bijv. `ztnfae7ktv`)**:

   ```bash
   sudo tcpdump -i ztnfae7ktv -w capture.pcap
   ```

3. **Stop opname**:
   `Ctrl+C`

## 📁 pcap-bestand kopiëren naar Windows

Vanuit Windows, gebruik:

```powershell
scp security@192.168.195.250:/security/capture.pcap C:\
```

> Zorg dat PowerShell als **administrator** draait bij gebruik van `scp`.
