# TOC (Table of Contents) - UI Improvement Ideas

## Dynamic Multi-Level TOC Rendering

**Bron**: [NoSysWhispers by klezvirus](https://klezvirus.github.io/posts/NoSysWhispers/)

### Concept

Een TOC die dynamisch meerdere dieptes rendert op het moment dat je voorbij een header scrollt.

### Voordelen

- Betere navigatie voor lange documenten
- Contextbehoud tijdens scrollen
- Verhoogde leesbaarheid en gebruiksvriendelijkheid

### Implementatie Overwegingen

- JavaScript-based scroll detection
- Adaptive rendering gebaseerd op viewport positie
- Performance impact bij zeer lange documenten
