## MIW Netwerekn

Welcome to Introduction to Computer Networks & Network Security.

An overview of this course can be found here in the Course Syllabus.

Students are introduced to computer networks and network security.

On completion of the course, students should be able to:

- Describe the terminology associated with computer networks.
- Identify the importance of the ISO 7-layer/IP -layer 5 reference models.
- Identify the key issues for the realization of the LAN/WAN/MAN network architectures and the hybridized existing form in the business environment and enterprise.
- Discuss the design principles of wired and wireless communication networks.
- Understand Cyber Risk Management Concepts & Threat Modelling.
- Discuss and explain current network authentication applications, PKI, Web & network security and their vulnerabilities that are exploited by intentional and unintentional attacks.
- Be able to describe network attacks and basic network defense tools.
