# Methodologies

## Frameworks for Information Security Management

- NIST Cybersecurity Framework
- ISO/IEC 27001
- COBIT (Control Objectives for Information and Related Technologies)
- ITIL (Information Technology Infrastructure Library)
- ISF (Information Security Forum) Standard of Good Practice for Information Security

## Risk Assessment and Management Frameworks

- FAIR (Factor Analysis of Information Risk)
- MITRE Shield
- CRAMM (CCTA Risk Analysis and Management Method)
- FAIR (Framework for Assessing and Improving Cybersecurity Resilience)

## Security Testing and Assessment Methodologies

- ISSAF (Information Systems Security Assessment Framework)
- OSSTMM (Open Source Security Testing Methodology Manual)
- PTES (Penetration Testing Execution Standard)
- CIS Controls (Center for Internet Security Controls)
- SANS Critical Security Controls
- CWE (Common Weakness Enumeration)
- CAPEC (Common Attack Pattern Enumeration and Classification)

## Threat Intelligence Frameworks

- MITRE ATT&CK Framework
- Cyber Kill Chain

## Web Application and Mobile Security Frameworks

- OWASP Top 10
- OWASP Mobile
- OWASP API

## Cloud and IoT Security Frameworks

- OWASP IoT
- CSA (Cloud Security Alliance) Security Guidance

## Compliance and Certification Standards

- CMMC (Cybersecurity Maturity Model Certification)
- ISO/IEC 15408 (Common Criteria)
- ITSG-33 (Information Technology Security Guidance)
- GDPR (General Data Protection Regulation)
- HIPAA (Health Insurance Portability and Accountability Act)
- PCI DSS (Payment Card Industry Data Security Standard)
- BSI Grundschutz (German Federal Office for Information Security Baseline Protection)

## Workforce and Role Frameworks

- NICE Cybersecurity Workforce Framework

## Incident Management and Incident Sharing Frameworks

- VERIS (Vocabulary for Event Recording and Incident Sharing)

## Defensive Security Frameworks

- DS-5 (Defensive Security Framework)

These frameworks and standards collectively provide organizations with a comprehensive toolkit for managing cybersecurity risks, complying with regulatory requirements, and building resilient security programs.
